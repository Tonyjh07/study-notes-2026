## 第十三章：集合框架

### ① 核心概念

**一句话总结：**
> 集合框架是Java提供的**容器类**，用来存储和操作一组对象，比数组更灵活、功能更丰富。

**生活类比：**
- **数组** = 固定大小的停车位（停满就不能再加）
- **集合** = 动态扩容的停车场（可以随时加车、找车、移车）

**用C的视角理解：**
集合框架就像C++的STL，封装了链表、动态数组、哈希表等数据结构，你不用自己实现。

---

### ② 为什么需要它？

**解决什么痛点？**

数组的局限性：
```java
// 数组的问题
String[] arr = new String[5];  // 固定大小
arr[0] = "A";
arr[1] = "B";
// 想加第3个元素？必须扩容，要自己写逻辑
// 想删除第1个元素？后面的要手动前移
// 想查找某个元素？要自己遍历
```

集合解决这些问题：
```java
// 集合的优势
List<String> list = new ArrayList<>();
list.add("A");     // 自动扩容
list.add("B");
list.add("C");      // 随时添加
list.remove(0);     // 自动前移
list.contains("B"); // 自动查找
```

---

### ③ 集合框架体系图

```
Collection（接口，存储单个元素）
├── List（有序、可重复）
│   ├── ArrayList（数组实现，查询快）
│   └── LinkedList（链表实现，增删快）
├── Set（无序、不可重复）
│   ├── HashSet（哈希表实现）
│   └── TreeSet（红黑树实现，自动排序）
└── Queue（队列）
    └── LinkedList

Map（接口，存储键值对）
├── HashMap（哈希表，无序）
└── TreeMap（红黑树，按key排序）
```

**一句话记忆：**
> List管顺序，Set管唯一，Map管键值对。

## 第一部分：List

### ① ArrayList（重点）

**核心概念：** 动态数组，查询快、增删慢（尾部增删快）。

```java
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        // 创建
        List<String> list = new ArrayList<>();
        
        // 添加
        list.add("苹果");    // ["苹果"]
        list.add("香蕉");    // ["苹果", "香蕉"]
        list.add(1, "橙子"); // ["苹果", "橙子", "香蕉"]
        
        // 获取
        String fruit = list.get(1);  // "橙子"
        
        // 删除
        list.remove(0);      // 删除索引0 → ["橙子", "香蕉"]
        list.remove("香蕉"); // 删除对象 → ["橙子"]
        
        // 遍历方式1：普通for
        for (int i = 0; i < list.size(); i++) {
            System.out.println(list.get(i));
        }
        
        // 遍历方式2：增强for（推荐）
        for (String s : list) {
            System.out.println(s);
        }
        
        // 其他常用方法
        list.size();              // 大小
        list.contains("橙子");    // true，是否包含
        list.isEmpty();          // false，是否为空
        list.clear();            // 清空
    }
}
```

---

### ② LinkedList

**核心概念：** 双向链表，增删快（头部/中间）、查询慢。

```java
import java.util.LinkedList;

LinkedList<String> list = new LinkedList<>();
list.add("A");           // 尾部添加
list.addFirst("头");     // 头部添加
list.addLast("尾");      // 尾部添加
String first = list.getFirst();  // 获取头部
String last = list.getLast();    // 获取尾部
list.removeFirst();      // 删除头部
list.removeLast();       // 删除尾部
```

---

### ③ ArrayList vs LinkedList

| 操作 | ArrayList | LinkedList |
|------|-----------|------------|
| 尾部添加 | O(1) 快 | O(1) 快 |
| 头部添加 | O(n) 慢 | O(1) 快 |
| 中间添加 | O(n) 慢 | O(1)（定位O(n)） |
| 随机访问 | O(1) 快 | O(n) 慢 |
| 内存占用 | 连续内存 | 分散内存 + 指针 |

**选择原则：**
```java
// 查询多、尾部添加 → ArrayList
List<String> list = new ArrayList<>();

// 头部/中间增删多 → LinkedList
List<String> list = new LinkedList<>();
```

## 第二部分：Set

### ① HashSet（重点）

**核心概念：** 元素**不可重复**，**无序**，基于HashMap实现。

```java
import java.util.HashSet;
import java.util.Set;

Set<String> set = new HashSet<>();
set.add("苹果");
set.add("香蕉");
set.add("苹果");  // 添加失败，返回false

System.out.println(set);  // [香蕉, 苹果]（无序）
System.out.println(set.size());  // 2

// 判断是否包含
if (set.contains("苹果")) {
    System.out.println("有苹果");
}

// 遍历
for (String s : set) {
    System.out.println(s);
}
```

---

### ② TreeSet

**核心概念：** 自动**排序**（升序），基于TreeMap实现。

```java
import java.util.TreeSet;

TreeSet<Integer> set = new TreeSet<>();
set.add(5);
set.add(1);
set.add(8);
set.add(3);
System.out.println(set);  // [1, 3, 5, 8] 自动排序
```

## 第三部分：Map

### ① HashMap（重点）

**核心概念：** 存储**键值对**，通过键快速找到值，键唯一，无序。

```java
import java.util.HashMap;
import java.util.Map;

public class Main {
    public static void main(String[] args) {
        // 创建
        Map<String, Integer> scores = new HashMap<>();
        
        // 添加/修改（键存在则覆盖）
        scores.put("小明", 85);
        scores.put("小红", 92);
        scores.put("小明", 90);  // 覆盖，现在小明=90
        
        // 获取
        Integer score = scores.get("小红");  // 92
        Integer notExist = scores.get("小李"); // null
        
        // 判断
        boolean hasKey = scores.containsKey("小明");  // true
        boolean hasValue = scores.containsValue(90); // true
        
        // 删除
        scores.remove("小红");
        
        // 遍历方式1：遍历所有键
        for (String name : scores.keySet()) {
            System.out.println(name + ":" + scores.get(name));
        }
        
        // 遍历方式2：遍历所有键值对（推荐）
        for (Map.Entry<String, Integer> entry : scores.entrySet()) {
            System.out.println(entry.getKey() + ":" + entry.getValue());
        }
        
        // 其他方法
        scores.size();      // 大小
        scores.isEmpty();   // 是否为空
        scores.clear();     // 清空
    }
}
```
