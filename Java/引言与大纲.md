## 引言：为什么学习Java？

### 一、Java的地位与应用

**一句话概括：**
> Java是企业级后端开发、Android应用、大数据平台的首选语言之一，全球拥有超过1200万开发者。

**Java能做什么？**

| 领域 | 应用场景 | 网络工程结合点 |
|------|----------|----------------|
| **后端开发** | 电商、金融、社交平台 | 服务器端逻辑、API接口 |
| **网络编程** | HTTP服务器、聊天室 | TCP/IP协议实现、Socket通信 |
| **大数据** | Hadoop、Spark | 海量数据处理、分布式计算 |
| **Android** | 手机App | 移动端网络请求 |
| **物联网** | 智能设备 | 设备间通信协议 |

### 二、Java的核心优势

| 优势 | 说明 | 对比C语言 |
|------|------|-----------|
| **跨平台** | JVM屏蔽操作系统差异 | C程序需针对不同平台重新编译 |
| **面向对象** | 代码复用、维护性更好 | C是面向过程 |
| **自动内存管理** | GC垃圾回收，不用手动free | C需手动管理内存 |
| **丰富的生态** | 海量开源库、框架 | C生态集中在系统层 |
| **安全性** | 强类型检查、异常处理、安全管理器 | C更依赖程序员经验 |

### 三、Java的挑战

| 挑战 | 说明 |
|------|------|
| JVM启动开销 | 相比C/C++启动稍慢 |
| 内存占用 | JVM运行时占用更多内存 |
| 底层控制 | 不如C/C++直接操作硬件 |

### 四、学习目标

完成本课程后，你将能够：

1. **独立开发**：从零到一完成Java控制台应用、桌面GUI程序
2. **网络编程**：实现Socket通信、HTTP客户端/服务器
3. **理解原理**：掌握JVM内存模型、多线程机制、集合底层实现
4. **项目实战**：完成聊天室、HTTP服务器等网络相关项目
5. **持续进阶**：具备学习Spring Boot、分布式框架的基础
