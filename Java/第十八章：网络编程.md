## 第十八章：网络编程

### ① 核心概念

**一句话总结：**
> Java网络编程通过`Socket`（套接字）实现网络通信——客户端和服务端通过IP地址+端口号建立连接，传输数据。

**生活类比：**
- **IP地址** = 学校地址（找到哪台计算机）
- **端口号** = 收件人姓名（找到哪个应用程序）
- **Socket** = 电话线（建立连接的通路）

**用C的视角理解：**
相当于C语言的socket库（`socket()`、`bind()`、`listen()`、`accept()`），但Java封装成更简洁的面向对象API。

---

### ② 为什么需要网络编程？

**解决什么痛点？**

让不同计算机上的程序能够通信——Web服务器、聊天软件、在线游戏都依赖它。

**核心作用：**
1. **跨机器通信**：不局限于单机程序
2. **分布式系统**：多台服务器协同工作
3. **数据传输**：文件下载、视频流、API调用

---

### ③ TCP vs UDP（重点对比）

| 特性 | TCP | UDP |
|------|-----|-----|
| 连接 | 面向连接（三次握手） | 无连接 |
| 可靠性 | 可靠，保证数据到达 | 不可靠，可能丢包 |
| 顺序 | 保证顺序 | 不保证 |
| 速度 | 较慢 | 快 |
| 场景 | HTTP、FTP、文件传输 | 视频直播、DNS、游戏 |
| Java类 | Socket / ServerSocket | DatagramSocket |

**一句话选择：**
> 文件要完整 → TCP；视频可卡顿 → UDP

## TCP编程

### ④ 服务端（ServerSocket）

```java
import java.io.*;
import java.net.*;

public class TCPServer {
    public static void main(String[] args) throws IOException {
        // 1. 创建ServerSocket，绑定端口
        ServerSocket server = new ServerSocket(8888);
        System.out.println("服务端启动，等待连接...");
        
        // 2. 接受客户端连接（阻塞）
        Socket socket = server.accept();
        System.out.println("客户端已连接：" + socket.getInetAddress());
        
        // 3. 接收消息
        BufferedReader reader = new BufferedReader(
            new InputStreamReader(socket.getInputStream())
        );
        String msg = reader.readLine();
        System.out.println("收到：" + msg);
        
        // 4. 发送响应
        PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);
        writer.println("消息已收到：" + msg);
        
        // 5. 关闭
        socket.close();
        server.close();
    }
}
```

### ⑤ 客户端（Socket）

```java
import java.io.*;
import java.net.*;

public class TCPClient {
    public static void main(String[] args) throws IOException {
        // 1. 连接服务端（IP + 端口）
        Socket socket = new Socket("127.0.0.1", 8888);
        
        // 2. 发送消息
        PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);
        writer.println("你好，服务器！");
        
        // 3. 接收响应
        BufferedReader reader = new BufferedReader(
            new InputStreamReader(socket.getInputStream())
        );
        String response = reader.readLine();
        System.out.println("服务端响应：" + response);
        
        // 4. 关闭
        socket.close();
    }
}
```

**运行顺序：先启动服务端，再启动客户端**

### ⑥ 多线程服务端（结合多线程知识）

```java
public class MultiThreadServer {
    public static void main(String[] args) throws IOException {
        ServerSocket server = new ServerSocket(8888);
        System.out.println("服务端启动...");
        
        while (true) {
            Socket socket = server.accept();  // 阻塞等待新连接
            // 每个客户端单独开一个线程处理
            new Thread(() -> handleClient(socket)).start();
        }
    }
    
    private static void handleClient(Socket socket) {
        try {
            BufferedReader reader = new BufferedReader(
                new InputStreamReader(socket.getInputStream())
            );
            PrintWriter writer = new PrintWriter(socket.getOutputStream(), true);
            
            String msg;
            while ((msg = reader.readLine()) != null) {
                System.out.println("收到：" + msg);
                writer.println("服务器回复：" + msg);
            }
            socket.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
```

## UDP编程

### ⑦ 服务端（DatagramSocket）

```java
import java.net.*;

public class UDPServer {
    public static void main(String[] args) throws Exception {
        // 1. 创建DatagramSocket，绑定端口
        DatagramSocket socket = new DatagramSocket(8888);
        byte[] buffer = new byte[1024];
        System.out.println("UDP服务端启动...");
        
        while (true) {
            // 2. 接收数据包
            DatagramPacket packet = new DatagramPacket(buffer, buffer.length);
            socket.receive(packet);  // 阻塞
            
            // 3. 解析数据
            String msg = new String(packet.getData(), 0, packet.getLength());
            System.out.println("收到：" + msg);
            
            // 4. 发送响应（可选）
            byte[] response = ("已收到：" + msg).getBytes();
            DatagramPacket responsePacket = new DatagramPacket(
                response, response.length,
                packet.getAddress(), packet.getPort()
            );
            socket.send(responsePacket);
        }
    }
}
```

### ⑧ 客户端（DatagramSocket）

```java
import java.net.*;

public class UDPClient {
    public static void main(String[] args) throws Exception {
        // 1. 创建DatagramSocket
        DatagramSocket socket = new DatagramSocket();
        
        // 2. 发送数据
        String msg = "你好，UDP服务器！";
        byte[] data = msg.getBytes();
        DatagramPacket packet = new DatagramPacket(
            data, data.length,
            InetAddress.getByName("127.0.0.1"), 8888
        );
        socket.send(packet);
        System.out.println("发送：" + msg);
        
        // 3. 接收响应
        byte[] buffer = new byte[1024];
        DatagramPacket responsePacket = new DatagramPacket(buffer, buffer.length);
        socket.receive(responsePacket);
        String response = new String(responsePacket.getData(), 0, responsePacket.getLength());
        System.out.println("服务端响应：" + response);
        
        socket.close();
    }
}
```

### ⑨ URL和HttpURLConnection

```java
import java.io.*;
import java.net.*;

public class URLDemo {
    public static void main(String[] args) throws Exception {
        // 访问网页
        URL url = new URL("https://www.example.com");
        HttpURLConnection conn = (HttpURLConnection) url.openConnection();
        conn.setRequestMethod("GET");
        
        // 读取响应
        BufferedReader reader = new BufferedReader(
            new InputStreamReader(conn.getInputStream())
        );
        String line;
        while ((line = reader.readLine()) != null) {
            System.out.println(line);
        }
        reader.close();
        conn.disconnect();
    }
}
```
