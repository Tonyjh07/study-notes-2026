## 补充章节：图形用户界面入门（Java Swing）

### ① 核心概念

**一句话总结：**
> Java Swing是Java的桌面图形界面工具包，通过组件（窗口、按钮、文本框）和事件监听机制，让用户能通过鼠标点击、键盘输入等方式与程序交互。

**生活类比：**
- **控制台程序** = 对讲机（只能发命令、看文字）
- **GUI程序** = 智能手机（点按钮、滑动、看到丰富界面）

**为什么学Swing？**
聊天室程序需要输入框（打字）、按钮（发送）、文本区域（显示聊天记录），这些都需要GUI。

---

### ② Java GUI 体系简图

```
AWT（抽象窗口工具包，重量级组件，依赖本地系统）
└── Swing（轻量级组件，纯Java实现，跨平台）
    ├── JFrame        （窗口/主框架）
    ├── JPanel        （面板，用于布局组织）
    ├── JButton       （按钮）
    ├── JTextField    （单行文本框）
    ├── JTextArea     （多行文本区域）
    ├── JLabel        （标签，显示文字/图片）
    ├── JScrollPane   （滚动条）
    ├── JList         （列表）
    └── JMenuBar      （菜单栏）
```

**一句话：** Swing是Java自带的GUI库，无需额外安装。

### ③ 为什么需要GUI？

**解决什么痛点？**

控制台程序交互不友好：

```java
// 控制台聊天：全靠输入命令
System.out.println("输入消息：");
String msg = scanner.nextLine();
// 看不到历史记录，体验差
```

GUI让聊天更直观：
- 显示消息记录（滚动区域）
- 独立输入框
- 点击发送按钮

### ④ 第一个Swing程序

```java
import javax.swing.*;

public class FirstSwing {
    public static void main(String[] args) {
        // 1. 创建窗口（JFrame）
        JFrame frame = new JFrame("我的第一个窗口");
        frame.setSize(400, 300);        // 设置宽高
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);  // 关闭窗口时退出程序
        
        // 2. 创建标签
        JLabel label = new JLabel("Hello, Swing!");
        label.setHorizontalAlignment(SwingConstants.CENTER);
        
        // 3. 添加到窗口
        frame.add(label);
        
        // 4. 显示窗口
        frame.setVisible(true);
    }
}
```

### ⑤ 聊天室界面核心组件

```java
import javax.swing.*;
import java.awt.*;

public class ChatRoomGUI {
    public static void main(String[] args) {
        // 主窗口
        JFrame frame = new JFrame("聊天室");
        frame.setSize(500, 400);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        
        // 设置布局：BorderLayout（东西南北中五区域）
        frame.setLayout(new BorderLayout());
        
        // 1. 显示消息区域（中间）
        JTextArea chatArea = new JTextArea();
        chatArea.setEditable(false);           // 只读，不能编辑
        chatArea.setLineWrap(true);             // 自动换行
        JScrollPane scrollPane = new JScrollPane(chatArea);  // 加滚动条
        frame.add(scrollPane, BorderLayout.CENTER);
        
        // 2. 底部区域（南）：输入框 + 发送按钮
        JPanel bottomPanel = new JPanel(new BorderLayout());
        
        JTextField inputField = new JTextField();     // 输入框
        JButton sendButton = new JButton("发送");      // 发送按钮
        
        bottomPanel.add(inputField, BorderLayout.CENTER);
        bottomPanel.add(sendButton, BorderLayout.EAST);
        frame.add(bottomPanel, BorderLayout.SOUTH);
        
        // 3. 顶部区域（北）：显示在线用户列表（可选）
        JList<String> userList = new JList<>(new String[]{"用户1", "用户2", "用户3"});
        JScrollPane userScroll = new JScrollPane(userList);
        userScroll.setPreferredSize(new Dimension(120, 0));  // 固定宽度
        frame.add(userScroll, BorderLayout.WEST);
        
        frame.setVisible(true);
        
        // 事件处理见下一节
    }
}
```

**布局说明：**
```
┌────────────────────────────────────────┐
│  [在线用户列表]    聊天消息显示区域      │
│                    (JTextArea)          │
│                    (可滚动)              │
│                                          │
├────────────────────────────────────────┤
│  [输入框 JTextField]        [发送按钮]  │
└────────────────────────────────────────┘
```

### ⑥ 事件监听（核心）

**一句话总结：**
> 事件监听让组件（按钮、输入框）对用户操作（点击、打字）做出响应。

```java
import javax.swing.*;
import java.awt.event.*;

public class EventDemo {
    public static void main(String[] args) {
        JFrame frame = new JFrame("事件示例");
        frame.setSize(400, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setLayout(new FlowLayout());
        
        JButton button = new JButton("点我");
        JLabel label = new JLabel("等待点击");
        
        // 方式1：匿名内部类（传统写法）
        button.addActionListener(new ActionListener() {
            @Override
            public void actionPerformed(ActionEvent e) {
                label.setText("按钮被点击了！");
            }
        });
        
        // 方式2：Lambda表达式（简洁，推荐）
        // button.addActionListener(e -> label.setText("按钮被点击了！"));
        
        frame.add(button);
        frame.add(label);
        frame.setVisible(true);
    }
}
```

**聊天室常用事件：**
```java
// 发送按钮点击
sendButton.addActionListener(e -> {
    String msg = inputField.getText();
    if (!msg.trim().isEmpty()) {

// 输入框按回车键发送

});
```

### ⑦ 线程与GUI（重要）

**一句话总结：**
> Swing是单线程模型，所有界面更新必须在**事件分发线程（EDT）** 中执行，网络等耗时任务要在后台线程做。

```java
// ❌ 错误：在EDT中做网络操作（界面会卡死）
button.addActionListener(e -> {
    // 这行在EDT中执行，会阻塞界面
    String response = socket.receive();  // 可能等很久
    chatArea.append(response);
});

// ✅ 正确：后台线程接收消息，再交给EDT更新界面
button.addActionListener(e -> {
    new Thread(() -> {  // 后台线程
        String response = socket.receive();  // 不阻塞界面
        // 更新界面要切回EDT
        SwingUtilities.invokeLater(() -> {
            chatArea.append(response);
        });
    }).start();
});
```

### ⑧ 聊天室程序实现逻辑

```
客户端（GUI）                    服务端
    │                              │
    │  1. 连接服务器                │
    │─────────────────────────────→│
    │                              │
    │  2. 输入消息，点发送          │
    │─────────────────────────────→│  广播给所有客户端
    │                              │
    │  3. 接收其他用户消息（后台线程）
    │←─────────────────────────────│
    │                              │
    │  4. 更新聊天记录区域          │
    │    SwingUtilities.invokeLater
```

**核心类设计：**
```java
class ChatClient {
    private JFrame frame;
    private JTextArea chatArea;
    private JTextField inputField;
    private Socket socket;
    private PrintWriter writer;
    private BufferedReader reader;
    
    // 1. 初始化GUI
    private void initGUI() { ... }
    
    // 2. 连接服务器
    private void connectToServer(String host, int port) { ... }
    
    // 3. 发送消息（按钮事件）
    private void sendMessage(String msg) {
        writer.println(msg);
    }
    
    // 4. 接收消息（后台线程，循环监听）
    private void startReceiving() {
        new Thread(() -> {
            String msg;
            while ((msg = reader.readLine()) != null) {
                final String finalMsg = msg;
                SwingUtilities.invokeLater(() -> {
                    chatArea.append(finalMsg + "\n");
                });
            }
        }).start();
    }
}
```
