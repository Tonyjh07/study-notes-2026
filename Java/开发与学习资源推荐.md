# 第一部分：JDK与IDE选择

## ① JDK（Java Development Kit）

### 推荐版本
| 版本 | 推荐程度 | 说明 |
|------|----------|------|
| **JDK 25 (LTS)** | ⭐⭐⭐⭐ | 最新LTS版，更多语法特性 |
| **JDK 21 (LTS)** | ⭐⭐⭐⭐⭐ | 主流版本，稳定，推荐用于学习 |
| **JDK 17 (LTS)** | ⭐⭐⭐⭐⭐ | 热门版本，稳定，企业常用 |
| **JDK 8 (LTS)** | ⭐⭐⭐⭐ | 经典版本，很多老项目使用 |

> **建议：** 选 **JDK 21** 或 **JDK 17**，兼容性好，足够完成课程项目。

### 下载地址
| 来源 | 地址 | 说明 |
|------|------|------|
| **Oracle JDK** | https://www.oracle.com/java/technologies/downloads/ | 官方版，商业用途需授权 |
| **OpenJDK** | https://adoptium.net/ | 开源免费，最推荐 |
| **Amazon Corretto** | https://aws.amazon.com/corretto/ | 亚马逊版，免费，稳定 |

> **推荐：** Adoptium (Eclipse Temurin) —— 免费、开源、更新及时。

### 环境变量配置（Windows）
```bash
# JAVA_HOME
C:\Program Files\Eclipse Adoptium\jdk-17.0.9.9-hotspot

# Path 中添加
%JAVA_HOME%\bin

# 验证安装
java -version
javac -version
```

## ② IDE（集成开发环境）

| IDE | 推荐度 | 优点 | 缺点 | 适合人群 |
|-----|--------|------|------|----------|
| **IntelliJ IDEA** | ⭐⭐⭐⭐⭐ | 智能提示最强、重构强大、插件丰富 | 专业版收费（社区版免费够用） | 主力开发，强烈推荐 |
| **VS Code** | ⭐⭐⭐⭐ | 轻量、免费、插件生态好 | Java支持不如IDEA原生 | 轻量编辑，多语言开发 |
| **Eclipse** | ⭐⭐⭐ | 免费、历史悠久 | 界面老、体验一般 | 学校实验课常见 |
| **NetBeans** | ⭐⭐ | 免费 | 市场份额小 | 不推荐 |

### 推荐方案
```text
日常学习/项目开发：IntelliJ IDEA Community Edition（免费）
快速修改/查看代码：VS Code + Java插件包
学校上机考试：Eclipse（提前熟悉环境）
```

### 下载地址
- **IntelliJ IDEA Community**：https://www.jetbrains.com/idea/download/
- **VS Code**：https://code.visualstudio.com/
- **Eclipse**：https://www.eclipse.org/downloads/

# 第五部分：小建议

### 给网络工程专业同学的建议

1. **结合专业学Java**：
   - 多线程 → 服务器并发处理
   - Socket → TCP/UDP协议实现
   - 异常处理 → 网络容错设计

2. **动手比看更重要**：
   - 每个知识点写3-5个小例子
   - 遇到报错先自己分析，再搜索解决

3. **善用工具**：
   - IDE调试器比`System.out.println`强大10倍
   - Git每天用，养成提交习惯

4. **多看开源代码**：
   - GitHub搜索 "Java socket server" 等关键词
   - 学习别人的代码风格和架构
