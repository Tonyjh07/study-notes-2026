# 离散数学·集合论 补充篇 —— 关系闭包、偏序关系与函数


## 一、本章知识地图（补充部分）

在上一部分学习了关系的基本概念和五种性质（自反、反自反、对称、反对称、传递）之后，自然会产生一个问题：**如果给定的关系不具备某种性质，能否通过添加最少量的序偶使它具备该性质？** 这就引出了**关系的闭包**概念。闭包本质上是“补全”操作——自反闭包补上所有缺失的自环，对称闭包补上所有缺失的逆向边，传递闭包补上所有缺失的传递边。其中传递闭包最为复杂，涉及关系的幂运算和Warshall算法。

闭包运算完成后，我们继续深入关系的分类研究。根据关系满足的性质组合不同，可以得到两类最重要的特殊关系：**等价关系**（已在上一部分介绍）和**偏序关系**。偏序关系是自反、反对称、传递的关系，它刻画了“大小”“先后”“包含”等具有方向性的比较结构。偏序集上的**哈斯图**是一种经过简化的关系图——省略自环、省略箭头方向（默认向上）、省略由传递性可推出的边，只保留“覆盖”关系。在偏序集中，我们需要研究各种**特殊元素**：极大元/极小元、最大元/最小元、上界/下界、上确界/下确界。如果偏序集中任意两个元素都有最小上界和最大下界，则称为**格**。

关系理论的最终落脚点是**函数**。函数是一种特殊的关系——每个自变量都有**唯一**的像。从关系角度看，函数就是满足“单值性”的二元关系。函数进一步按性质分为**单射**（不同的自变量对应不同的因变量）、**满射**（值域等于整个陪域）和**双射**（既是单射又是满射）。函数的**复合运算**和**逆运算**本质上就是关系的复合与逆运算在函数上的特殊情形。函数的基数理论还告诉我们：可以用双射来比较集合的大小——存在双射的两个集合等势。


## 二、核心知识点详解

### 2.1 关系的闭包（Closure）

**· 概念名称：** 关系的闭包

**· 通俗解释：** 闭包就是“补齐”操作。好比一个社交网络中，如果要求每个人都认识自己（自反性），但当前关系不具备这个性质，我们就给每个人添加一条“认识自己”的关系——添加得尽可能少，刚好满足要求为止。

**· 严格定义：** 设R是集合A上的关系，若存在A上的关系R′满足：（1）R⊆R′；（2）R′具有性质P（自反/对称/传递）；（3）对于任何具有性质P且包含R的关系S，都有R′⊆S（即R′是**最小**的），则称R′是R关于性质P的**闭包**。

**· 三种闭包记号：** r(R)表示自反闭包，s(R)表示对称闭包，t(R)表示传递闭包。


**· 概念名称：** 自反闭包（Reflexive Closure）—— r(R)

**· 通俗解释：** 给关系中的每个顶点都加上一个指向自己的“自环”。好比一个班级的花名册，原来只有“同学”关系，现在要求每个人都“认识自己”。

**· 严格定义：** r(R) = R ∪ I_A，其中 I_A = {〈a,a〉 | a∈A} 是A上的恒等关系。

**· 推导思路：** 自反闭包要求每个元素都有自环。如果R中已有某些自环，就保留；缺少哪些，就补上哪些。用R∪I_A正好做到“最少添加”——只添加了所有缺失的自环。

**· 具体数值例子：**
A = {1, 2, 3}，R = {〈1,2〉,〈2,3〉}
I_A = {〈1,1〉,〈2,2〉,〈3,3〉}
r(R) = R ∪ I_A = {〈1,2〉,〈2,3〉,〈1,1〉,〈2,2〉,〈3,3〉}

**· 使用条件：** 必须明确集合A（因为 I_A 依赖于A）。


**· 概念名称：** 对称闭包（Symmetric Closure）—— s(R)

**· 通俗解释：** 如果a指向b有一条边，就补上b指向a的边。好比“关注”关系——如果甲关注了乙，但乙没有关注甲，为了满足对称性，就补上乙关注甲。

**· 严格定义：** s(R) = R ∪ R⁻¹，其中 R⁻¹ = {〈b,a〉 | 〈a,b〉∈R} 是R的逆关系。

**· 推导思路：** 对称闭包要求每个有序对〈a,b〉都有对应的〈b,a〉。用R∪R⁻¹正好做到——对每条边都补上反向边，已有的双向边保持不变。

**· 具体数值例子：**
A = {1, 2, 3}，R = {〈1,2〉,〈2,3〉}
R⁻¹ = {〈2,1〉,〈3,2〉}
s(R) = R ∪ R⁻¹ = {〈1,2〉,〈2,1〉,〈2,3〉,〈3,2〉}

**· 使用条件：** 无特殊条件。


**· 概念名称：** 传递闭包（Transitive Closure）—— t(R)

**· 通俗解释：** 如果从a能经过若干步到达c，就补上a直接到c的边。好比“祖先”关系——如果a是b的祖先，b是c的祖先，那么a也是c的祖先。

**· 严格定义：** t(R) = R ∪ R² ∪ R³ ∪ …，其中 R² = R∘R（关系的复合），R³ = R²∘R，依此类推。

**· 推导思路：** 传递闭包要求：只要有路径从a到b（长度≥1），就应有直接的有序对〈a,b〉。R的k次幂正好表示长度为k的路径。把所有长度的路径都并起来，就得到传递闭包。对于有限集合，只需计算到Rⁿ即可（后面会循环重复）。

**· 具体数值例子：**
A = {1, 2, 3, 4}，R = {〈1,2〉,〈2,3〉,〈3,4〉}
R² = {〈1,3〉,〈2,4〉}
R³ = {〈1,4〉}
R⁴ = ∅
t(R) = R ∪ R² ∪ R³ = {〈1,2〉,〈2,3〉,〈3,4〉,〈1,3〉,〈2,4〉,〈1,4〉}

**· 使用条件：** 对于有限集合A，传递闭包可以通过有限次幂运算得到。


**· 概念名称：** Warshall算法（求传递闭包的矩阵算法）

**· 通俗解释：** 当集合元素较多时，逐次计算R的幂太麻烦。Warshall算法是一种高效的矩阵方法，通过动态规划思想，依次允许经过第1个、第2个……第n个顶点作为“中转站”，逐步扩充关系矩阵。

**· 算法步骤：**
设关系矩阵为M（n×n，0/1矩阵），M[i][j]=1表示〈i,j〉∈R。
1. 初始化矩阵M
2. 对于 k = 1 到 n（k表示允许经过的中间顶点编号）：
   对于 i = 1 到 n：
     对于 j = 1 到 n：
        如果 M[i][k] = 1 且 M[k][j] = 1，则 M[i][j] = 1
3. 最终M即为传递闭包的矩阵

**· 具体数值例子：**
A = {1, 2, 3}，R = {〈1,2〉,〈2,3〉}
初始矩阵（行→列）：
```
   1 2 3
1 [0 1 0]
2 [0 0 1]
3 [0 0 0]
```
k=1（允许经过顶点1）：无变化
k=2（允许经过顶点2）：M[1][2]=1且M[2][3]=1 ⇒ M[1][3]=1
k=3（允许经过顶点3）：无变化
最终矩阵：
```
   1 2 3
1 [0 1 1]
2 [0 0 1]
3 [0 0 0]
```
对应 t(R) = {〈1,2〉,〈1,3〉,〈2,3〉}


**· 闭包的重要定理：**
- **定理1**：R自反 ⇔ r(R)=R；R对称 ⇔ s(R)=R；R传递 ⇔ t(R)=R
- **定理2**：若 R₁⊆R₂，则 r(R₁)⊆r(R₂)，s(R₁)⊆s(R₂)，t(R₁)⊆t(R₂)
- **定理3**：r(R₁∪R₂)=r(R₁)∪r(R₂)；s(R₁∪R₂)=s(R₁)∪s(R₂)；t(R₁∪R₂)⊇t(R₁)∪t(R₂)


### 2.2 偏序关系（Partial Order Relation）

**· 概念名称：** 偏序关系与偏序集

**· 通俗解释：** 偏序关系就是用来比较“大小”或“先后”的关系，但这种比较不要求任意两个元素都能比较。好比“包含关系”——{1}⊆{1,2}可以比较，但{1,2}和{2,3}谁包含谁？都不能，它们不可比。

**· 严格定义：** 集合A上的关系R，若同时满足：（1）**自反性**（∀x∈A，xRx）；（2）**反对称性**（若xRy且yRx，则x=y）；（3）**传递性**（若xRy且yRz，则xRz），则称R为A上的**偏序关系**，通常记作 ≤。称(A, ≤)为**偏序集**。

**· 具体数值例子：**
A = {1, 2, 3, 4, 6, 12}，定义 a≤b 当且仅当 a|b（a整除b）。
验证：自反（a|a）、反对称（若a|b且b|a则a=b）、传递（若a|b且b|c则a|c）。
所以整除关系是偏序关系。


**· 概念名称：** 可比与全序

**· 通俗解释：** 在偏序集中，如果两个元素能分出大小，就称它们“可比”；如果任意两个元素都可比，就称为“全序”。

**· 严格定义：** 在偏序集(A, ≤)中，若 a≤b 或 b≤a，则称a与b**可比**。若A中任意两个元素都可比，则称≤为**全序关系**（或线序关系）。

**· 具体数值例子：**
- 整除关系在{1,2,4,8}上是全序（1|2|4|8，任意两个可比）
- 整除关系在{1,2,3,6}上是全序
- 整除关系在{1,2,3,4,6,12}上**不是**全序（2和3不可比，4和6不可比）


**· 概念名称：** 覆盖与哈斯图（Hasse Diagram）

**· 通俗解释：** 哈斯图是偏序关系的一种简化画法。它去掉自环、去掉箭头（默认向上）、去掉由传递性可推出的边，只保留“覆盖”关系。

**· 严格定义：** 在偏序集(A, ≤)中，若 x<y（即x≤y且x≠y），且不存在z∈A使得 x<z<y，则称 **y覆盖x**。哈斯图只画出覆盖关系：下方元素≤上方元素，连线表示覆盖。

**· 具体数值例子：**
A = {1, 2, 3, 4, 6, 12}，≤为整除关系。
覆盖关系：

- 1覆盖？1<2（无中间数）✓；1<3✓；1<4？有1<2<4，所以4不覆盖1
- 2覆盖？2<4✓；2<6✓；2<12？有2<4<12或2<6<12，所以12不覆盖2
- 3覆盖？3<6✓；3<12？有3<6<12，所以12不覆盖3
- 4覆盖？4<12✓
- 6覆盖？6<12✓
哈斯图：（2和6之间应有连线）
```
        12
       /  \
      4    6
      |    |
      2    3
       \  /
        1
```


**· 概念名称：** 极大元、极小元

**· 通俗解释：** 极大元就是“没有比它更大的元素”——在哈斯图中处在最顶层的元素。极小元就是“没有比它更小的元素”——在哈斯图中处在最底层的元素。

**· 严格定义：**
- **极大元**：x是极大元 ⇔ 对任意y∈A，若 x≤y，则 x=y
- **极小元**：x是极小元 ⇔ 对任意y∈A，若 y≤x，则 x=y

**· 具体数值例子（同上哈斯图）：**
极大元：只有12（没有比12更大的）
极小元：只有1（没有比1更小的）


**· 概念名称：** 最大元、最小元

**· 通俗解释：** 最大元是“比所有元素都大”的元素——所有元素都≤它。最小元是“比所有元素都小”的元素——它≤所有元素。

**· 严格定义：**
- **最大元**：x是最大元 ⇔ 对任意y∈A，y≤x
- **最小元**：x是最小元 ⇔ 对任意y∈A，x≤y

**· 与极大/极小元的区别：** 最大元必须和**所有**元素可比（都比它小），而极大元只需要“没有比它更大的”——可以有和它不可比的元素。

**· 具体数值例子：**
A = {1, 2, 3, 4, 6, 12}，整除关系：
最大元：12（所有元素都整除12）✓
最小元：1（1整除所有元素）✓


**· 概念名称：** 上界、下界、上确界、下确界

**· 通俗解释：** 对于集合B中的元素，上界就是“比B中所有元素都大”的元素（可以在B外）。上确界是所有上界中最小的那个。

**· 严格定义（设B⊆A）：**
- **上界**：a是B的上界 ⇔ 对任意b∈B，b≤a
- **下界**：a是B的下界 ⇔ 对任意b∈B，a≤b
- **上确界（最小上界）** ：B的所有上界中的最小元
- **下确界（最大下界）** ：B的所有下界中的最大元

**· 具体数值例子（同上哈斯图，B={2,3}）：**
- B的上界：能被2和3同时整除的元素 → 6, 12（都是上界）
- B的上确界：6（最小的上界）
- B的下界：同时整除2和3的元素 → 1
- B的下确界：1


**· 概念名称：** 格（Lattice）

**· 通俗解释：** 如果偏序集中任意两个元素都有唯一的最小上界和最大下界，就称为格。好比整除关系中，任意两个数都有最小公倍数（上确界）和最大公约数（下确界）。

**· 严格定义：** 偏序集(L, ≤)中，若任意两个元素a,b都有最小上界（记为a∨b，称为**并**）和最大下界（记为a∧b，称为**交**），则称(L, ≤)为**格**。

**· 具体数值例子：**
A = {1, 2, 3, 4, 6, 12}，整除关系构成格：
- 2∨3 = lcm(2,3) = 6，2∧3 = gcd(2,3) = 1
- 4∨6 = lcm(4,6) = 12，4∧6 = gcd(4,6) = 2


### 2.3 函数（Function）

**· 概念名称：** 函数的基本定义

**· 通俗解释：** 函数就是从集合A到集合B的一种“对应规则”——A中的每个元素都对应到B中**唯一**的一个元素。好比每个学生（定义域）都对应一个学号（值域），而且每个学生只有一个学号。

**· 严格定义：** 设A、B是集合，f是A到B的二元关系（f⊆A×B）。若对每个x∈A，都存在**唯一**的y∈B使得〈x,y〉∈f，则称f是A到B的**函数**（或**映射**），记作 f: A→B。

**· 关键术语：**
- A称为**定义域**（domain），记作 dom(f)=A
- B称为**陪域**（codomain）
- f(A)= { f(x) | x∈A } ⊆ B 称为**值域**（range）
- 对x∈A，唯一对应的y称为x的**像**（image）；对y∈B，满足f(x)=y的所有x称为y的**原像**（preimage）

**· 判断是否为函数的两个条件：**
1. **存在性**：A中每个元素都有像
2. **唯一性**：每个x只对应一个y

**· 具体数值例子：**
- f: {1,2,3} → {a,b,c}，f(1)=a，f(2)=b，f(3)=c ✓ 是函数
- f: {1,2,3} → {a,b,c}，f(1)=a，f(1)=b（同一个x对应两个y）✗ 不是函数（违反唯一性）
- f: {1,2} → {a,b,c}，f(1)=a ✗ 不是函数（2没有像，违反存在性）


**· 概念名称：** 单射（Injection / One-to-One）

**· 通俗解释：** 不同的自变量一定对应不同的因变量——不会有两个不同的人有同一个学号。

**· 严格定义：** 函数 f: A→B 称为**单射**，若对任意 x₁,x₂∈A，当 f(x₁)=f(x₂) 时，必有 x₁=x₂。等价地：x₁≠x₂ ⇒ f(x₁)≠f(x₂)。

**· 必要条件：** |A| ≤ |B|。


**· 概念名称：** 满射（Surjection / Onto）

**· 通俗解释：** B中的每个元素都至少被A中某个元素对应到——值域等于整个陪域。

**· 严格定义：** 函数 f: A→B 称为**满射**，若 ran(f)=B。即对任意 y∈B，都存在 x∈A 使得 f(x)=y。

**· 必要条件：** |B| ≤ |A|。


**· 概念名称：** 双射（Bijection / One-to-One Correspondence）

**· 通俗解释：** 既是单射又是满射——A和B的元素一一对应。

**· 严格定义：** 函数 f: A→B 称为**双射**，若 f 既是单射又是满射。

**· 必要条件：** |A| = |B|。

**· 重要应用：** 存在双射的两个集合**等势**（基数相同），这是比较无限集合大小的基础。


**· 概念名称：** 特殊函数

**· 常数函数：** f: A→B，存在某个 b∈B，使得对所有 x∈A，f(x)=b。

**· 恒等函数：** I_A: A→A，I_A(x)=x。

**· 特征函数：** χ_A: E→{0,1}，χ_A(x)=1 ⇔ x∈A。


**· 概念名称：** 函数的复合（Composition）

**· 通俗解释：** 先执行一个函数，再执行另一个函数——把两个函数“串联”起来。

**· 严格定义：** 设 f: A→B，g: B→C（即 ran(f)⊆dom(g)），则 g 对 f 的**复合函数** g∘f: A→C 定义为：(g∘f)(x) = g(f(x))。（国内教程为f(g(x))）

**· 重要性质：**

- 若 f、g 都是单射，则 g∘f 也是单射
- 若 f、g 都是满射，则 g∘f 也是满射
- 若 f、g 都是双射，则 g∘f 也是双射

**· 概念名称：** 逆函数（Inverse Function）

**· 通俗解释：** 逆函数就是把映射方向反过来——从B回到A。但并不是所有函数都有逆函数，只有双射才有。

**· 严格定义：** 若 f: A→B 是双射，则 f 的**逆关系** f⁻¹: B→A 也是函数，称为 f 的**逆函数**。f⁻¹(y)=x 当且仅当 f(x)=y。

**· 为什么只有双射才有逆函数：**
- 若f不是单射，则存在x₁≠x₂但f(x₁)=f(x₂)=y，逆关系中y对应两个值，违反唯一性
- 若f不是满射，则存在y∈B没有原像，违反存在性


## 三、典型题型与完整解法

### 题型一：求关系的各种闭包

**· 题目特征：** 给出一个集合A上的关系R（用列举法或关系矩阵），要求求r(R)、s(R)或t(R)。

**· 标准解题步骤：**
1. 明确集合A和关系R
2. **自反闭包**：R ∪ I_A（给每个顶点加自环）
3. **对称闭包**：R ∪ R⁻¹（每条边补反向边）
4. **传递闭包**：可用两种方法——
   - 幂集法：R ∪ R² ∪ R³ ∪ …
   - Warshall算法（矩阵法）
5. 检查结果是否满足对应性质

**· 完整例题：**
> **题干：** 设 A = {1, 2, 3, 4}，R = {〈1,2〉,〈2,3〉,〈3,4〉}。求 r(R)、s(R)、t(R)。

**解答：**
- **r(R)** = R ∪ I_A = {〈1,2〉,〈2,3〉,〈3,4〉,〈1,1〉,〈2,2〉,〈3,3〉,〈4,4〉}
- **s(R)** = R ∪ R⁻¹，R⁻¹ = {〈2,1〉,〈3,2〉,〈4,3〉}
  s(R) = {〈1,2〉,〈2,1〉,〈2,3〉,〈3,2〉,〈3,4〉,〈4,3〉}
- **t(R)** = R ∪ R² ∪ R³
  R² = {〈1,3〉,〈2,4〉}
  R³ = {〈1,4〉}
  t(R) = {〈1,2〉,〈2,3〉,〈3,4〉,〈1,3〉,〈2,4〉,〈1,4〉}

**· 常见错误提醒：** 求传递闭包时不要遗漏间接路径；自反闭包要添加**所有**缺失的自环，不能漏掉任何一个。


### 题型二：画哈斯图并求特殊元素

**· 题目特征：** 给出一个偏序集（通常用整除关系或包含关系），要求画哈斯图，并求极大/极小元、最大/最小元、上/下界等。

**· 标准解题步骤：**
1. 先确定偏序关系（如整除、包含等）
2. 找出所有**覆盖关系**（x<y且中间无其他元素）
3. 画哈斯图：下方画小元素，上方画大元素，覆盖关系连线
4. 从图中读取特殊元素：
   - **极大元**：最顶层（没有向上的边）
   - **极小元**：最底层（没有向下的边）
   - **最大元**：唯一且在最顶层的元素（所有元素都≤它）
   - **最小元**：唯一且在最底层的元素（它≤所有元素）
   - **上界**：在B中所有元素上方的元素
   - **下界**：在B中所有元素下方的元素

**· 完整例题：**
> **题干：** 设 A = {1, 2, 3, 4, 6, 8, 12, 24}，≤为整除关系。画出哈斯图，求极大元、极小元、最大元、最小元。设B={4,6}，求B的上界、下界、上确界、下确界。

**解答：**
- 覆盖关系：
  1覆盖2, 3
  2覆盖4, 6
  3覆盖6
  4覆盖8, 12
  6覆盖12
  8覆盖24
  12覆盖24
- 哈斯图：
```
          24
         /  \
        8   12
        |   / \
        4  6  /
         \ /  /
          2  /
          \ /
           1
          （3在1的上方、6的下方，单独支路）
```
  更准确的图：
```
          24
         /  \
        8   12
        |  /  \
        4 /    6
        |/    / \
        2    3  /
         \  /  /
           1  /
```
- **极大元**：24（没有比它更大的）
- **极小元**：1（没有比它更小的）
- **最大元**：24（所有元素都整除24）
- **最小元**：1（1整除所有元素）
- B={4,6}的上界：能被4和6同时整除的元素 → 12, 24
- B={4,6}的下界：同时整除4和6的元素 → 1, 2
- B的上确界（最小上界）：12
- B的下确界（最大下界）：2

**· 常见错误提醒：** 极大元可以有多个，最大元最多只有一个；上界可以在B外部；不要把上确界和上界混淆——上确界是**最小**的上界。


### 题型三：判断函数类型（单射、满射、双射）

**· 题目特征：** 给出一个函数 f: A→B（用公式、列举法或映射图），要求判断是单射、满射还是双射。

**· 标准解题步骤：**
1. 明确定义域A和陪域B
2. **判断单射**：检查是否存在 x₁≠x₂ 但 f(x₁)=f(x₂)。若存在则不是单射
3. **判断满射**：检查B中每个元素是否都有原像。若有元素没有原像则不是满射
4. 若同时满足单射和满射 ⇒ 双射
5. 利用基数辅助判断：|A|>|B|时不可能单射；|A|<|B|时不可能满射

**· 完整例题：**
> **题干：** 判断以下函数是否为单射、满射、双射：
> （1）f: Z→Z，f(x)=2x
> （2）f: Z→Z，f(x)=x+1
> （3）f: Z→N，f(x)=|x|

**解答：**
- （1）f(x)=2x
  - 单射：若2x₁=2x₂ ⇒ x₁=x₂ ✓ 是单射
  - 满射：是否存在x使2x=3？x=1.5∉Z ✗ 不是满射
  - 双射：不是（缺满射）
- （2）f(x)=x+1
  - 单射：若x₁+1=x₂+1 ⇒ x₁=x₂ ✓ 是单射
  - 满射：对任意y∈Z，取x=y-1∈Z，有f(x)=y ✓ 是满射
  - 双射：是（单射+满射）
- （3）f: Z→N，f(x)=|x|
  - 单射：f(1)=1，f(-1)=1，1≠-1但像相同 ✗ 不是单射
  - 满射：对任意y∈N，取x=y（或x=-y），有f(x)=y ✓ 是满射
  - 双射：不是（缺单射）

**· 常见错误提醒：** 判断满射时要看陪域B是否**完全等于**值域，而不是看值域是否“看起来很大”；判断单射时要注意定义域中不同的元素可能映射到相同的值。


### 题型四：函数的复合与逆函数

**· 题目特征：** 给出两个函数 f 和 g，要求求复合函数 g∘f，或判断 f 是否有逆函数并求 f⁻¹。

**· 标准解题步骤：**
1. **复合函数**：确认 ran(f)⊆dom(g)，然后 (g∘f)(x)=g(f(x))
2. **逆函数**：先判断f是否为双射
   - 若不是双射 → 没有逆函数
   - 若是双射 → f⁻¹(y)=x 当且仅当 f(x)=y

**· 完整例题：**
> **题干：** 设 f: R→R，f(x)=2x+1；g: R→R，g(x)=x²。求 g∘f 和 f∘g。f是否有逆函数？

**解答：**
- g∘f: (g∘f)(x) = g(f(x)) = g(2x+1) = (2x+1)² = 4x²+4x+1
- f∘g: (f∘g)(x) = f(g(x)) = f(x²) = 2x²+1
- 注意：g∘f ≠ f∘g（函数复合一般不满足交换律）
- f是否有逆函数：f(x)=2x+1是双射（单射：2x₁+1=2x₂+1⇒x₁=x₂；满射：任意y，取x=(y-1)/2）
- f⁻¹(y) = (y-1)/2，即 f⁻¹(x) = (x-1)/2

**· 常见错误提醒：** 复合函数的顺序很重要——g∘f是先f后g；求逆函数前必须先确认原函数是双射。


## 四、易混淆概念对比表

### 对比组1：自反闭包 vs 对称闭包 vs 传递闭包

| 对比维度       | 自反闭包 r(R)              | 对称闭包 s(R)         | 传递闭包 t(R)            |
| -------------- | -------------------------- | --------------------- | ------------------------ |
| **定义**       | 包含R的最小自反关系        | 包含R的最小对称关系   | 包含R的最小传递关系      |
| **公式**       | R∪I_A                      | R∪R⁻¹                 | R∪R²∪R³∪…                |
| **关系图操作** | 每个顶点加自环             | 单向边变双向边        | 补上所有可达路径         |
| **添加的内容** | 只添加自环〈a,a〉          | 只添加反向边〈b,a〉   | 添加所有间接可达的有序对 |
| **计算复杂度** | O(n)                       | O(n²)                 | O(n³)（Warshall算法）    |
| **适用场景**   | 需要“每个元素和自己有关系” | 需要“关系是对称的”    | 需要“关系是传递的”       |
| **记忆技巧**   | “自己环”——r=reflexive      | “反向边”——s=symmetric | “传递路径”——t=transitive |

### 对比组2：极大元 vs 最大元

| 对比维度                      | 极大元（Maximal Element） | 最大元（Greatest Element） |
| ----------------------------- | ------------------------- | -------------------------- |
| **定义**                      | 没有比它更大的元素        | 比所有元素都大的元素       |
| **与所有元素可比？**          | 不需要（可以有不可比的）  | 必须与所有元素可比         |
| **唯一性**                    | 可以有多个                | 最多只有一个               |
| **存在性**                    | 有限偏序集必有            | 不一定存在                 |
| **例子（整除，A={2,3,5}）**   | 2、3、5都是极大元         | 不存在最大元               |
| **例子（整除，A={1,2,4,8}）** | 8是极大元                 | 8是最大元                  |
| **记忆技巧**                  | “极大”=局部最高点         | “最大”=全局最高点          |

### 对比组3：单射 vs 满射 vs 双射

| 对比维度         | 单射（Injection）        | 满射（Surjection）   | 双射（Bijection）        |
| ---------------- | ------------------------ | -------------------- | ------------------------ |
| **核心含义**     | 不同x对应不同y           | 每个y都有原像        | 一一对应                 |
| **定义公式**     | x₁≠x₂⇒f(x₁)≠f(x₂)        | ran(f)=B             | 单射+满射                |
| **基数条件**     | \|A\|≤\|B\|              | \|B\|≤\|A\|          | \|A\|=\|B\|              |
| **是否有逆函数** | 否（除非也是满射）       | 否（除非也是单射）   | 是                       |
| **图示理解**     | 没有两个箭头指向同一个点 | 每个点都有箭头指向它 | 每个点恰有一个箭头指向它 |
| **例子（A→B）**  | f(x)=2x（Z→Z）           | f(x)=\|x\|（Z→N）    | f(x)=x+1（Z→Z）          |
| **记忆技巧**     | “单”=一对一              | “满”=覆盖整个陪域    | “双”=两个条件都满足      |


## 五、本章小结与自测（补充部分）

### 一句话总结

> **闭包、偏序与函数的统一思想**：闭包是通过“最少添加”让关系获得指定性质；偏序是用自反+反对称+传递来刻画“比较”结构；函数是满足单值性的特殊关系——三者共同构成了从一般关系到特殊关系的完整理论体系。

### 自测题（补充）

#### 【概念辨析题】

**题1：** 判断下列说法是否正确：“任何函数都有逆函数。”

**答案：** 错误。只有**双射**函数才有逆函数。如果函数不是单射，则逆关系中一个y可能对应多个x，违反函数的唯一性；如果函数不是满射，则有些y没有原像，违反函数的存在性。

**题2：** 判断下列说法是否正确：“在偏序集中，极大元一定是最大元。”

**答案：** 错误。极大元只是“没有比它更大的元素”，但可能与其他元素不可比。最大元要求“比**所有**元素都大”，必须与所有元素可比。例如整除关系中{2,3,5}，2、3、5都是极大元，但没有最大元。

**答案：**
- r(R) = {〈1,2〉,〈2,3〉,〈1,1〉,〈2,2〉,〈3,3〉}
- s(R) = {〈1,2〉,〈2,1〉,〈2,3〉,〈3,2〉}
- t(R) = {〈1,2〉,〈2,3〉,〈1,3〉}

**题4：** 判断 f: R→R，f(x)=x² 是否为单射、满射、双射。

**答案：**
- 单射？否。f(1)=1，f(-1)=1，1≠-1但像相同。
- 满射？否。ran(f)=[0,+∞)≠R（负数没有原像）。
- 双射？否（单射和满射都不满足）。
