# 离散数学·数理逻辑 详细学习指南


## 1. 本章知识地图

数理逻辑是离散数学的基石，其核心任务是**将自然语言中的推理转化为符号系统，从而可以用数学方法研究推理的有效性**。整个知识体系按照“从简单到复杂、从语法到语义”的逻辑层层展开。

**第一步：认识命题。** 数理逻辑的起点是“命题”——一个能判断真假的陈述句。命题是逻辑推理的最小单位，好比化学中的原子。但单个命题无法表达复杂的推理关系，于是需要“联结词”来将多个命题组合成复合命题。五个基本联结词（否定¬、合取∧、析取∨、蕴含→、等价↔）定义了命题之间的逻辑运算规则，每个联结词都有严格的真值表来规定其真假条件。

**第二步：从命题到公式。** 将命题变元和联结词按规则组合，就得到“命题公式”（也称合式公式）。但公式有不同类型：有些在所有赋值下都为真（重言式/永真式），有些在所有赋值下都为假（矛盾式/永假式），其余则为可满足式。为了判定公式的类型，引入“真值表法”——列出所有可能的赋值并逐行计算真值，这是最直接但可能繁琐的方法。

**第三步：化简与标准化。** 真值表法虽然直观，但当命题变元增多时（n个变元就有2ⁿ行），计算量呈指数增长。为此引入“等值演算”——利用逻辑等值式（如德摩根律、分配律等）对公式进行等价变形，化简为更简单的形式。进一步地，为了给每个公式一个“标准身份证”，引入“范式”概念：析取范式（DNF）和合取范式（CNF），以及更严格的主析取范式与主合取范式。主范式的核心思想是：每个命题公式都唯一对应一组“成真赋值”或“成假赋值”，主析取范式由所有成真赋值对应的小项析取而成，主合取范式由所有成假赋值对应的大项合取而成。

**第四步：从命题到谓词。** 命题逻辑的局限性在于无法处理命题内部的结构（如“所有人都会死”中的“所有”）。于是引入“谓词逻辑”——将原子命题拆解为“个体词”（主语）和“谓词”（谓语/性质），并引入“量词”（全称量词∀和存在量词∃）来表达数量的概念。这大大增强了表达能力。

**第五步：谓词公式的标准化。** 谓词逻辑同样需要标准形式。“前束范式”要求所有量词移到公式最前面，辖域覆盖整个公式。“Skolem标准形”则进一步消去所有存在量词（用Skolem函数或常量替代），为自动推理（如归结原理）做准备。

**第六步：推理理论。** 最终目标是进行有效的逻辑推理。命题逻辑有一套推理规则（假言推理、拒取式、析取三段论等）；谓词逻辑则增加了量词相关的消去与引入规则（UI、UG、EI、EG）。这些规则构成了从前提推出结论的形式化系统。


## 2. 核心知识点详解

### 2.1 命题与命题变元

**· 通俗解释**
命题就是一句“能判断真假的话”。比如“今天下雨了”——这句话要么是真的，要么是假的，不会既真又假。而“今天天气真好！”不是命题，因为它是感叹句，无法判断真假。

**· 严格定义**
命题（Proposition）是具有确定真值的陈述句。一个命题的真值要么为真（True，记作T或1），要么为假（False，记作F或0），二者必居其一且只居其一。

命题变元（Propositional Variable）是用大写字母（如P、Q、R）表示任意命题的符号，其真值可以在T和F之间变化。

**· 推导思路**
判断一个句子是否为命题，只需检验两条：（1）是不是陈述句？（2）有没有确定的真假？满足这两条就是命题，否则不是。注意：暂时不知道真假（如“哥德巴赫猜想”）仍是命题，因为它有确定真假，只是尚未被证明。

**· 具体数值例子**
- “2 + 3 = 5”是真命题 ✅
- “2 + 3 = 6”是假命题 ✅
- “请把门关上”不是命题（祈使句）❌
- “这道题难吗？”不是命题（疑问句）❌

**· 使用条件/前提假设**
命题必须为陈述句且具有确定的真值。悖论（如“这句话是假的”）不是命题，因为它无法确定真假。


### 2.2 五种基本联结词

**（1）否定联结词 ¬（非）**

**· 通俗解释**
“否定”就是“不是”“并非”。如果P表示“今天下雨”，那么¬P表示“今天没下雨”。

**· 严格定义**
设P为命题，¬P（读作“非P”）称为P的否定。若P为真则¬P为假；若P为假则¬P为真。

**真值表：**

| P    | ¬P   |
| ---- | ---- |
| T    | F    |
| F    | T    |

**· 具体数值例子**
P：“北京是中国的首都”（真），则¬P：“北京不是中国的首都”（假）。

**· 使用条件/前提假设**
P必须是命题。


**（2）合取联结词 ∧（且）**

**· 通俗解释**
“合取”就是“并且”“同时”。P∧Q为真当且仅当P和Q同时为真。就像“我饿了并且有吃的”——只有两个条件都满足才算数。

**· 严格定义**
设P、Q为命题，P∧Q（读作“P且Q”）称为合取式。P∧Q为真当且仅当P和Q都为真。

**真值表：**

| P    | Q    | P∧Q  |
| ---- | ---- | ---- |
| F    | F    | F    |
| F    | T    | F    |
| T    | F    | F    |
| T    | T    | T    |

**· 具体数值例子**
P：“2是偶数”（真），Q：“3是奇数”（真），则P∧Q：“2是偶数且3是奇数”为真。
P：“2是偶数”（真），Q：“3是偶数”（假），则P∧Q为假。

**· 使用条件/前提假设**
P和Q都必须是命题。


**（3）析取联结词 ∨（或）**

**· 通俗解释**
“析取”就是“或者”。P∨Q为真当且仅当P和Q至少有一个为真。注意：数理逻辑中的“或”是“相容或”——允许两者同时为真（与自然语言中“要么…要么…”的“排斥或”不同）。

**· 严格定义**
设P、Q为命题，P∨Q（读作“P或Q”）称为析取式。P∨Q为假当且仅当P和Q都为假。

**真值表：**

| P    | Q    | P∨Q  |
| ---- | ---- | ---- |
| F    | F    | F    |
| F    | T    | T    |
| T    | F    | T    |
| T    | T    | T    |

**· 具体数值例子**
P：“2是偶数”（真），Q：“3是偶数”（假），则P∨Q：“2是偶数或3是偶数”为真。

**· 使用条件/前提假设**
P和Q都必须是命题。注意与自然语言“或”的区别——当说“你要么选A要么选B”时，通常表示排斥或，需要用(P∨Q)∧¬(P∧Q)来表示。


**（4）蕴含联结词 →（如果…那么…）**

**· 通俗解释**
“蕴含”表示条件关系。P→Q读作“如果P那么Q”。这里P称为前件（前提），Q称为后件（结论）。关键理解：**只有前件真而后件假时，蕴含式为假**；当前件为假时，无论后件真假，蕴含式都为真。这被称为“善意的推定”——“如果前提不成立，那随便说什么都行”。

**· 严格定义**
设P、Q为命题，P→Q（读作“P蕴含Q”或“如果P则Q”）称为蕴含式。P→Q为假当且仅当P为真且Q为假。

**真值表：**

| P    | Q    | P→Q  |
| ---- | ---- | ---- |
| F    | F    | T    |
| F    | T    | T    |
| T    | F    | F    |
| T    | T    | T    |

**· 具体数值例子**
P：“明天下雨”，Q：“比赛取消”。P→Q：“如果明天下雨，那么比赛取消”。只有明天下雨但比赛没取消时，这句话才是假的。

**· 使用条件/前提假设**
P和Q都必须是命题。注意：P→Q并不要求P和Q之间有因果联系，只关注真值之间的关系。


**（5）等价联结词 ↔（当且仅当）**

**· 通俗解释**
“等价”表示两个命题同真同假。P↔Q读作“P当且仅当Q”。P和Q要么同时为真，要么同时为假。

**· 严格定义**
设P、Q为命题，P↔Q（读作“P等价于Q”）称为等价式（双条件式）。P↔Q为真当且仅当P和Q的真值相同。

**真值表：**

| P    | Q    | P↔Q  |
| ---- | ---- | ---- |
| F    | F    | T    |
| F    | T    | F    |
| T    | F    | F    |
| T    | T    | T    |

**· 具体数值例子**
P：“三角形是等边三角形”，Q：“三角形是等角三角形”。P↔Q：“三角形是等边的当且仅当它是等角的”——真。

**· 使用条件/前提假设**
P和Q都必须是命题。


### 2.3 命题公式与公式类型

**· 通俗解释**
命题公式就是用命题变元和联结词按照规则组成的“表达式”。就像代数中用数字和运算符组成代数式一样。

**· 严格定义**
命题公式（Propositional Formula）的递归定义：
（1）单个命题变元是公式；
（2）若A是公式，则(¬A)是公式；
（3）若A和B是公式，则(A∧B)、(A∨B)、(A→B)、(A↔B)都是公式；
（4）有限次应用以上规则得到的符号串才是公式。

**公式类型：**
- **重言式（永真式，Tautology）** ：在所有赋值下都为真的公式。
- **矛盾式（永假式，Contradiction）** ：在所有赋值下都为假的公式。
- **可满足式（Satisfiable）** ：至少存在一组赋值使其为真的公式。

**· 具体数值例子**
公式：(P→Q)↔(¬P∨Q)
- 当P=T、Q=T时：T↔T = T
- 当P=T、Q=F时：F↔F = T
- 当P=F、Q=T时：T↔T = T
- 当P=F、Q=F时：T↔T = T
所有情况都为真，所以是重言式。


### 2.4 真值表法

**· 通俗解释**
真值表法就是列出命题变元所有可能的真假组合，然后逐行计算整个公式的真假。好比把所有可能的情况都列出来检查一遍。

**· 严格定义**
对于含有n个命题变元的公式，有2ⁿ种不同的赋值组合。真值表列出所有这些组合及对应的公式真值。

**· 具体数值例子**
求公式(P→Q)∧P的真值表：

| P    | Q    | P→Q  | (P→Q)∧P |
| ---- | ---- | ---- | ------- |
| F    | F    | T    | F       |
| F    | T    | T    | F       |
| T    | F    | F    | F       |
| T    | T    | T    | T       |

只有P和Q都为真时公式才为真。

**· 使用条件/前提假设**
公式中的联结词必须是已定义的五种基本联结词。


### 2.5 等值演算与基本等值式

**· 通俗解释**
等值演算就是用已知的等值式对公式进行等价变形，就像代数中用公式化简表达式一样。

**· 严格定义**
若等价式A↔B是重言式，则称A与B等值（逻辑等价），记作A⇔B。这意味着A和B在所有赋值下真值完全相同。

**基本等值式（16条核心定律）** ：

| 定律名称       | 公式                                         |
| -------------- | -------------------------------------------- |
| 双重否定律     | ¬¬A ⇔ A                                      |
| 幂等律         | A∧A ⇔ A，A∨A ⇔ A                             |
| 交换律         | A∧B ⇔ B∧A，A∨B ⇔ B∨A                         |
| 结合律         | (A∧B)∧C ⇔ A∧(B∧C)，(A∨B)∨C ⇔ A∨(B∨C)         |
| 分配律         | A∧(B∨C) ⇔ (A∧B)∨(A∧C)，A∨(B∧C) ⇔ (A∨B)∧(A∨C) |
| 德摩根律       | ¬(A∧B) ⇔ ¬A∨¬B，¬(A∨B) ⇔ ¬A∧¬B               |
| 吸收律         | A∨(A∧B) ⇔ A，A∧(A∨B) ⇔ A                     |
| 零律           | A∨T ⇔ T，A∧F ⇔ F                             |
| 同一律         | A∨F ⇔ A，A∧T ⇔ A                             |
| 排中律         | A∨¬A ⇔ T                                     |
| 矛盾律         | A∧¬A ⇔ F                                     |
| 蕴含等值式     | A→B ⇔ ¬A∨B                                   |
| 等价等值式     | A↔B ⇔ (A→B)∧(B→A) ⇔ (A∧B)∨(¬A∧¬B)            |
| 假言易位       | A→B ⇔ ¬B→¬A                                  |
| 等价否定等值式 | A↔B ⇔ ¬A↔¬B                                  |
| 归谬论         | (A→B)∧(A→¬B) ⇔ ¬A                            |

**· 具体数值例子**
证明 ¬(P∨Q) ⇔ ¬P∧¬Q（德摩根律）：
用真值表验证——当P=T、Q=T时，左边¬T=F，右边F∧F=F；当P=T、Q=F时，左边¬T=F，右边F∧T=F；其他情况同理，两边完全一致。

**· 使用条件/前提假设**
等值式中的A、B可以是任意命题公式。


### 2.6 范式（析取范式与合取范式）

**· 通俗解释**
“范式”就像是给公式规定一种“标准写法”。析取范式是“若干合取项的析取”（像若干个“且”用“或”连起来）；合取范式是“若干析取项的合取”（像若干个“或”用“且”连起来）。

**· 严格定义**
- **文字（Literal）** ：命题变元或其否定（如P、¬P）。
- **简单合取式**：若干文字的合取（如P∧¬Q∧R）。
- **简单析取式**：若干文字的析取（如P∨¬Q∨R）。
- **析取范式（DNF）** ：简单合取式的析取（如(P∧Q)∨(¬P∧R)）。
- **合取范式（CNF）** ：简单析取式的合取（如(P∨Q)∧(¬P∨R)）。

**求范式的步骤** ：
（1）消去→和↔（用蕴含等值式和等价等值式）；
（2）将¬移到命题变元前（用德摩根律和双重否定律）；
（3）用分配律展开。

**· 具体数值例子**
求(P→Q)∧R的合取范式：
- P→Q ⇔ ¬P∨Q
- 原式 ⇔ (¬P∨Q)∧R
已经是合取范式（两个简单析取式的合取）。

**· 使用条件/前提假设**
公式必须是合式公式。


### 2.7 主析取范式与主合取范式

**· 通俗解释**
普通范式不唯一（同一个公式可以有不同的范式写法），但主范式是唯一的。主析取范式由“小项”（极小项）的析取组成，每个小项对应一组使公式为真的赋值。

**· 严格定义**
- **极小项（小项，Min term）** ：含有n个命题变元的简单合取式，每个变元恰好出现一次（以原形或否定形式）。极小项对应唯一一组赋值使其为真。
- **极大项（大项，Max term）** ：含有n个命题变元的简单析取式，每个变元恰好出现一次。极大项对应唯一一组赋值使其为假。
- **主析取范式**：所有使公式为真的赋值所对应的极小项的析取。
- **主合取范式**：所有使公式为假的赋值所对应的极大项的合取。

**求解方法**：
（1）**真值表法**：列出真值表，找出所有成真赋值→对应极小项析取即主析取范式；找出所有成假赋值→对应极大项合取即主合取范式。
（2）**等值演算法**：先得析取范式，再将缺项的简单合取式补齐（用P∨¬P恒为真的技巧）。

**· 具体数值例子**
求公式P→Q的主析取范式和主合取范式。

真值表：

| P    | Q    | P→Q  |
| ---- | ---- | ---- |
| F    | F    | T    |
| F    | T    | T    |
| T    | F    | F    |
| T    | T    | T    |

成真赋值：00、01、11 → 对应极小项：¬P∧¬Q、¬P∧Q、P∧Q
主析取范式：(¬P∧¬Q)∨(¬P∧Q)∨(P∧Q)

成假赋值：10 → 对应极大项：¬P∨Q
主合取范式：¬P∨Q

**· 使用条件/前提假设**
公式必须是合式公式。主范式具有唯一性，是公式的“标准身份证”。


### 2.8 谓词与量词

**· 通俗解释**
命题逻辑的原子命题不能再分，但“所有的人都是要死的”这句话里其实包含结构——“所有的人”和“是要死的”。谓词逻辑把命题拆成“个体”（人）和“谓词”（是要死的），并用量词表达“所有”或“存在”。

**· 严格定义**
- **个体词（Individual）** ：独立存在的客体，可以是具体事物或抽象概念。个体变元常用x、y、z表示。
- **个体域（Domain）** ：个体变元的取值范围。全总个体域指宇宙间一切事物。
- **谓词（Predicate）** ：表示个体性质或个体之间关系的词。用F(x)表示x具有性质F；用F(x,y)表示x和y具有关系F。
- **全称量词（Universal Quantifier）∀** ：表示“所有的”“每一个”“任意的”。∀xF(x)表示个体域中所有x都具有性质F。
- **存在量词（Existential Quantifier）∃** ：表示“存在”“至少有一个”“有的”。∃xF(x)表示个体域中存在x具有性质F。

**· 具体数值例子**
设个体域为“人的集合”，F(x)：x是会死的。
- ∀xF(x)：“所有人都会死”——真命题。
- ∃xF(x)：“有人会死”——真命题。
- ∃x¬F(x)：“有人不会死”——假命题（在现实世界中）。

**· 使用条件/前提假设**
必须明确个体域。量词的辖域（Scope）是指量词所作用的范围。


### 2.9 前束范式

**· 通俗解释**
前束范式就是把公式中所有的量词都“提到最前面”，让后面只剩下一个不含量词的“母式”。

**· 严格定义**
前束范式（Prenex Normal Form）的形式为：
(Q₁x₁)(Q₂x₂)…(Qₙxₙ)β
其中每个Qᵢ是∀或∃，β是不含量词的母式。

**求解步骤**：
（1）消去→和↔（用等值式）；
（2）将¬深入移到原子谓词前（用德摩根律和量词否定等值式）；
（3）约束变元换名（避免冲突）；
（4）将所有量词前移到公式最前端（用量词辖域扩张/收缩规则）；
（5）将母式化为合取或析取范式（可选）。

**量词否定等值式**：
- ¬∀xF(x) ⇔ ∃x¬F(x)
- ¬∃xF(x) ⇔ ∀x¬F(x)

**· 具体数值例子**
求¬∀xF(x)∨∃xG(x)的前束范式：
- ¬∀xF(x) ⇔ ∃x¬F(x)
- 原式 ⇔ ∃x¬F(x)∨∃xG(x)
- 换名（第二个x改为y）：∃x¬F(x)∨∃yG(y)
- 前移：∃x∃y(¬F(x)∨G(y))

**· 使用条件/前提假设**
公式必须是谓词逻辑的合式公式。换名时要注意不能改变量词的约束关系。


### 2.10 Skolem标准形

**· 通俗解释**
Skolem标准形是在前束范式的基础上，把所有存在量词都“干掉”——用函数或常量来代替被存在量词约束的变元。

**· 严格定义**
Skolem标准形是**只含全称量词**的前束范式。消去存在量词的方法：
- 若存在量词前面没有全称量词，用**Skolem常量**代替该变元；
- 若存在量词前面有全称量词∀x₁…∀xₙ，用**Skolem函数**f(x₁,…,xₙ)代替该变元。

**· 具体数值例子**
∀x∃yP(x,y)的Skolem标准形：
- 存在量词∃y前面有全称量词∀x，用Skolem函数y=f(x)代替y
- 得到：∀xP(x,f(x))

**· 使用条件/前提假设**
公式须先化为前束范式。Skolem标准形与原公式**不等价**（只保持** satisfiability（可满足性）** 相同），这一点很重要。


### 2.11 推理规则（命题逻辑）

**· 通俗解释**
推理规则就是“从已知的前提可以合法地得到什么结论”的规则。

**· 严格定义**
若A→B是重言式，则称从A可推出B，记作A⇒B。

**主要推理规则**：

| 规则名称   | 形式           | 说明                   |
| ---------- | -------------- | ---------------------- |
| 附加律     | A ⇒ A∨B        | 如果A真，则A或B真      |
| 化简律     | A∧B ⇒ A        | 如果A且B真，则A真      |
| 假言推理   | A，A→B ⇒ B     | 如果A真且A蕴含B，则B真 |
| 拒取式     | ¬B，A→B ⇒ ¬A   | 如果B假且A蕴含B，则A假 |
| 析取三段论 | A∨B，¬A ⇒ B    | 如果A或B真且A假，则B真 |
| 假言三段论 | A→B，B→C ⇒ A→C | 蕴含的传递性           |

**· 具体数值例子**
前提：如果下雨（P），则地面湿（Q）；下雨（P）。结论：地面湿（Q）。
形式：P→Q，P ⇒ Q（假言推理）。

**· 使用条件/前提假设**
前提必须为真。推理规则保证：前提真则结论必真。


### 2.12 推理规则（谓词逻辑）

**· 通俗解释**
谓词逻辑的推理除了命题逻辑的规则外，还需要处理量词——如何去掉量词使用具体实例，又如何从具体实例回到全称命题。

**· 严格定义**：

| 规则名称     | 缩写 | 形式          | 说明                                        |
| ------------ | ---- | ------------- | ------------------------------------------- |
| 全称量词消去 | UI   | ∀xA(x) ⇒ A(c) | 所有x都满足，则任意特定c也满足              |
| 全称量词引入 | UG   | A(y) ⇒ ∀xA(x) | 若任意y都满足（y任意选取），则所有x满足     |
| 存在量词消去 | EI   | ∃xA(x) ⇒ A(c) | 存在x满足，则可取一个特定c满足（c是新常元） |
| 存在量词引入 | EG   | A(c) ⇒ ∃xA(x) | 若特定c满足，则存在x满足                    |

**· 具体数值例子**
前提：∀x(人(x)→会死(x))，人(苏格拉底)。结论：会死(苏格拉底)。
- ∀x(人(x)→会死(x)) （前提）
- 人(苏格拉底)→会死(苏格拉底) （UI，用苏格拉底代入）
- 人(苏格拉底) （前提）
- 会死(苏格拉底) （假言推理）

**· 使用条件/前提假设**
EI中引入的常元c必须是新的（之前未出现过）；UG中y必须是任意选取的，不能有关于y的任何额外假设。


## 3. 典型题型与完整解法

### 题型一：命题符号化

**· 题目特征**
给出自然语言描述的一句话或一段话，要求用命题变元和联结词将其符号化。关键词包括“如果…那么…”“并且”“或者”“当且仅当”“除非”“只有…才…”等。

**· 标准解题步骤**
（1）识别句子中的原子命题，用字母（P、Q、R等）分别表示；
（2）分析句子结构，找出联结词对应的逻辑关系；
（3）注意自然语言与逻辑符号的转换技巧（如“除非P否则Q”⇔¬P→Q）；
（4）写出完整的符号化公式。

**· 完整例题**
**题干**：将下列命题符号化：“如果明天天气好并且我有时间，那么我就去公园；否则，我就待在家里。”

**解答**：
设 P：明天天气好；Q：我有时间；R：我去公园；S：我待在家里。
“如果天气好并且我有时间，那么我就去公园”：(P∧Q)→R
“否则”（即“如果并非(天气好且我有时间)”）：¬(P∧Q)→S
整体：((P∧Q)→R) ∧ (¬(P∧Q)→S)

**答案**：((P∧Q)→R) ∧ (¬(P∧Q)→S)

**· 常见错误提醒**
（1）混淆“或”的相容与排斥——“或者…或者…”在逻辑中通常是相容或；
（2）“只有P才Q”应符号化为Q→P（不是P→Q）；
（3）忽略“除非”的转换——“除非P否则Q”=¬P→Q。


### 题型二：真值表与公式类型判定

**· 题目特征**
给出一个命题公式，要求列出真值表，并判定公式是重言式、矛盾式还是可满足式。

**· 标准解题步骤**
（1）确定公式中命题变元的个数n，准备2ⁿ行；
（2）列出所有可能的真假组合（通常按二进制顺序）；
（3）从内到外逐列计算子公式的真值；
（4）观察最后一列：全为T→重言式；全为F→矛盾式；既有T又有F→可满足式。

**· 完整例题**
**题干**：用真值表判断公式(P→Q)↔(¬P∨Q)的类型。

**解答**：

| P    | Q    | ¬P   | P→Q  | ¬P∨Q | (P→Q)↔(¬P∨Q) |
| ---- | ---- | ---- | ---- | ---- | ------------ |
| F    | F    | T    | T    | T    | T            |
| F    | T    | T    | T    | T    | T            |
| T    | F    | F    | F    | F    | T            |
| T    | T    | F    | T    | T    | T            |

最后一列全为T，因此该公式是**重言式**（永真式）。

**答案**：重言式

**· 常见错误提醒**
（1）遗漏某些赋值组合（必须完整列出2ⁿ种）；
（2）蕴含式的真值容易算错（只有前件真后件假时为假）；
（3）混淆“重言式”和“可满足式”——只要有一行假就不是重言式。


### 题型三：等值演算与范式求解

**· 题目特征**
给出一个命题公式，要求用等值演算求其析取范式、合取范式、主析取范式或主合取范式。

**· 标准解题步骤**
（1）消去→和↔（用蕴含等值式、等价等值式）；
（2）将¬移到命题变元前（用德摩根律、双重否定律）；
（3）用分配律展开（求DNF用∧对∨分配；求CNF用∨对∧分配）；
（4）求主范式时，将缺项的简单合取/析取式补齐（用P∨¬P或P∧¬P）；
（5）化简并整理。

**· 完整例题**
**题干**：用等值演算求公式(P→Q)∧R的主析取范式。

**解答**：
(1) 消去→：P→Q ⇔ ¬P∨Q
原式 ⇔ (¬P∨Q)∧R

(2) 展开：⇔ (¬P∧R)∨(Q∧R)

(3) 补齐缺项（三个变元P、Q、R）：
- ¬P∧R 缺Q：¬P∧R ⇔ ¬P∧(Q∨¬Q)∧R ⇔ (¬P∧Q∧R)∨(¬P∧¬Q∧R)
- Q∧R 缺P：Q∧R ⇔ (P∨¬P)∧Q∧R ⇔ (P∧Q∧R)∨(¬P∧Q∧R)

(4) 合并：⇔ (¬P∧¬Q∧R)∨(¬P∧Q∧R)∨(P∧Q∧R)

**答案**：(¬P∧¬Q∧R)∨(¬P∧Q∧R)∨(P∧Q∧R)

**· 常见错误提醒**
（1）求主范式时忘记补齐所有变元；
（2）分配律使用方向错误；
（3）合并时遗漏重复项。


### 题型四：谓词逻辑符号化

**· 题目特征**
给出包含“所有”“有些”“每一个”“存在”等量词的自然语言命题，要求用谓词逻辑符号化。

**· 标准解题步骤**
（1）确定个体域（题目是否指定）；
（2）定义谓词（用F、G、H等表示性质和关系）；
（3）识别量词（∀还是∃）及其辖域；
（4）注意量词的嵌套顺序和否定形式；
（5）写出完整的谓词公式。

**· 完整例题**
**题干**：将下列命题符号化：“每个人都有至少一个朋友。”（个体域为人的集合）

**解答**：
设 F(x,y)：x是y的朋友。
“每个人都有至少一个朋友”⇔ 对于每个人x，都存在一个人y，使得y是x的朋友。
符号化：∀x∃y F(y,x)

**答案**：∀x∃y F(y,x)

**· 常见错误提醒**
（1）∀和∃的顺序很重要——∀x∃y和∃y∀x含义不同；
（2）注意谓词中参数的顺序（F(y,x)表示y是x的朋友）；
（3）量词的辖域要明确，必要时加括号。


### 题型五：逻辑推理证明

**· 题目特征**
给出一组前提和一个结论，要求用推理规则证明结论可以从前提推出。关键词：“证明”“推理”“结论”“前提”。

**· 标准解题步骤**
（1）将前提和结论符号化；
（2）列出所有前提；
（3）从前提出发，逐层应用推理规则（假言推理、拒取式、UI、EI等）；
（4）每一步注明使用的规则；
（5）最终得到结论。

**· 完整例题**
**题干**：证明：前提“∀x(人(x)→会死(x))”和“人(苏格拉底)”可推出“会死(苏格拉底)”。

**解答**：
(1) ∀x(人(x)→会死(x)) 【前提】
(2) 人(苏格拉底)→会死(苏格拉底) 【(1) UI，全称量词消去】
(3) 人(苏格拉底) 【前提】
(4) 会死(苏格拉底) 【(2)(3) 假言推理】

**答案**：得证。

**· 常见错误提醒**
（1）EI引入的常元必须是新的，不能与已有常元重名；
（2）UG要求被泛化的个体是任意选取的，不能有额外假设；
（3）推理时不能跳跃步骤，每一步都要有规则依据。


## 4. 易混淆概念对比表

### 对比组一：蕴含→ 与 等价↔

| 对比维度     | 蕴含（→）            | 等价（↔）                |
| ------------ | -------------------- | ------------------------ |
| **定义**     | “如果P那么Q”         | “P当且仅当Q”             |
| **真值条件** | 只有P真Q假时为假     | P和Q同真同假时为真       |
| **公式**     | P→Q ⇔ ¬P∨Q           | P↔Q ⇔ (P→Q)∧(Q→P)        |
| **日常对应** | “如果…就…”           | “当且仅当”“充分必要条件” |
| **对称性**   | 不对称（P→Q ≠ Q→P）  | 对称（P↔Q ⇔ Q↔P）        |
| **记忆技巧** | 记住“前真后假才为假” | 记住“同真同假即为真”     |

### 对比组二：析取∨（相容或）与 异或⊕（排斥或）

| 对比维度     | 析取（∨）            | 异或（⊕）                  |
| ------------ | -------------------- | -------------------------- |
| **定义**     | “P或Q”               | “要么P要么Q”               |
| **真值条件** | P和Q至少一个为真     | P和Q恰好一个为真           |
| **真值表**   | F∨F=F；其他为T       | T⊕T=F；T⊕F=T；F⊕T=T；F⊕F=F |
| **日常对应** | “或者…或者…”（可兼） | “要么…要么…”（不可兼）     |
| **公式关系** | 基本联结词           | P⊕Q ⇔ (P∨Q)∧¬(P∧Q)         |
| **记忆技巧** | 记住“有真即真”       | 记住“不同为真，相同为假”   |

### 对比组三：全称量词∀ 与 存在量词∃

| 对比维度     | 全称量词（∀）                   | 存在量词（∃）                   |
| ------------ | ------------------------------- | ------------------------------- |
| **含义**     | “所有的”“每一个”                | “存在”“至少有一个”              |
| **真值条件** | 个体域中所有个体都满足          | 个体域中至少一个个体满足        |
| **否定关系** | ¬∀xF(x) ⇔ ∃x¬F(x)               | ¬∃xF(x) ⇔ ∀x¬F(x)               |
| **日常对应** | “一切”“凡”“所有”                | “有的”“有些”“至少一个”          |
| **推理规则** | UI（消去）、UG（引入）          | EI（消去）、EG（引入）          |
| **记忆技巧** | 记住“∀要求全部，一个反例就推翻” | 记住“∃只需一个，找到一个就足够” |


## 5. 本章小结与自测

### 一句话总结

数理逻辑的核心思想是：**用符号化的形式语言精确表达命题和推理，通过真值表、等值演算和推理规则来判定公式的真假和推理的有效性。**


### 自测题

#### 概念辨析题（2道）

**题1**：判断以下说法是否正确并说明理由：“如果一个命题公式不是重言式，那么它一定是矛盾式。”

**答案**：❌ 错误。命题公式分为三类：重言式（永真）、矛盾式（永假）和可满足式（既非永真也非永假）。不是重言式的公式可能是矛盾式，也可能是可满足式（如P∧Q在某些赋值为真、某些为假）。

**题2**：判断以下说法是否正确并说明理由：“在谓词逻辑中，∀x∃yP(x,y)和∃y∀xP(x,y)是等值的。”

**答案**：❌ 错误。∀x∃yP(x,y)表示“对每个x，都存在一个y（可能依赖于x）使得P(x,y)成立”；而∃y∀xP(x,y)表示“存在一个固定的y，使得对所有x都有P(x,y)成立”。前者弱于后者。例如P(x,y)表示“x<y”，在实数域中∀x∃y(x<y)为真（对任意x总可以取更大的y），但∃y∀x(x<y)为假（不存在一个比所有实数都大的数）。


#### 简单计算题（2道）

**题3**：用真值表判断公式(P∧Q)→(P∨Q)是否为重言式。

**答案**：

| P    | Q    | P∧Q  | P∨Q  | (P∧Q)→(P∨Q) |
| ---- | ---- | ---- | ---- | ----------- |
| F    | F    | F    | F    | T           |
| F    | T    | F    | T    | T           |
| T    | F    | F    | T    | T           |
| T    | T    | T    | T    | T           |

最后一列全为T，因此是**重言式**。

**题4**：求公式¬(P→Q)的主析取范式。

**答案**：
(1) ¬(P→Q) ⇔ ¬(¬P∨Q) 【蕴含等值式】
(2) ⇔ P∧¬Q 【德摩根律】
(3) 这是含有两个变元的简单合取式，已经是一个极小项
主析取范式：P∧¬Q


#### 综合应用题（2道）

**题5**：将下列命题符号化并判断其真值（个体域为整数集合）：“存在一个整数x，使得对任意整数y，都有x ≤ y。”

**答案**：
设谓词L(x,y)：x ≤ y。
符号化：∃x∀y L(x,y)
即：∃x∀y (x ≤ y)
真值判断：存在一个整数x小于等于所有整数——这要求x是所有整数中的最小值。但整数集合没有最小值（可以无限小），因此该命题为**假**。

**题6**：证明：前提“如果P则Q”和“如果Q则R”可以推出“如果P则R”。

**解答**：
(1) P→Q 【前提】
(2) Q→R 【前提】
(3) 假设P 【附加前提，用于条件证明】
(4) Q 【(1)(3) 假言推理】
(5) R 【(2)(4) 假言推理】
(6) P→R 【(3)-(5) 条件证明（CP规则）】

得证：P→Q，Q→R ⇒ P→R（假言三段论）。
