# 第9章 光的偏振 详细自学笔记


## 1. 本章知识地图

光的偏振这一章的核心逻辑是：**从“光是横波”这一基本属性出发，逐步建立描述光振动方向特性的完整知识体系**。

**第一步：建立基本概念。** 光是一种电磁波，其中电场强度矢量 **E**（称为光矢量）与光的传播方向垂直。由于原子发光的独立性和随机性，普通光源发出的光中，光矢量在垂直于传播方向的平面内沿各个方向振动的几率均等，这种光称为**自然光**。自然光可以分解为两个互相垂直、振幅相等但无固定相位关系的线偏振光。

**第二步：引入“偏振”概念。** 如果光矢量只沿某一个固定方向振动，这种光称为**线偏振光**（也称平面偏振光）。介于自然光和线偏振光之间的是**部分偏振光**——各个方向都有振动，但不同方向上的强度不同。

**第三步：获得偏振光的方法引出两大定律。** 如何从自然光中获得线偏振光？主要有三种途径。途径一：利用偏振片的**二向色性**（对某一方向的光振动有选择吸收的特性），让自然光通过偏振片后变为线偏振光——这个过程引出**马吕斯定律**，定量描述线偏振光通过检偏器后光强随夹角的变化。途径二：利用光在界面上的**反射和折射**——当入射角等于特定值（布儒斯特角）时，反射光成为完全偏振光——这个过程引出**布儒斯特定律**。

**第四步：深入晶体的双折射现象。** 途径三：光通过某些各向异性晶体（如方解石）时会分裂为两束折射光——**寻常光（o光）**和**非常光（e光）** ——这称为**双折射现象**。o光遵守折射定律，振动方向垂直于光轴；e光不遵守折射定律，振动方向在光轴平面内。

**第五步：波片与偏振态的转换。** 从双折射晶体中切割出平行平面板称为**波晶片**（波片）。线偏振光通过波片后，o光和e光产生相位差。通过控制波片厚度，可以改变出射光的偏振态：**四分之一波片**（相位差π/2）可将线偏振光变为圆偏振光或椭圆偏振光；**半波片**（相位差π）可使线偏振光的振动方向旋转。

**第六步：偏振态的丰富。** 当线偏振光通过波片且振动方向与光轴成45°时，若波片为四分之一波片，出射光为**圆偏振光**；若成其他角度，出射光为**椭圆偏振光**。圆偏振光和椭圆偏振光的光矢量末端在垂直于传播方向的平面上的轨迹分别为圆和椭圆。

**贯穿全章的检验方法：** 用偏振片旋转可以区分不同偏振态——旋转一周，光强无变化的是自然光或圆偏振光；有变化且有消光（完全变暗）的是线偏振光；有变化但不消光的是部分偏振光或椭圆偏振光。


## 2. 核心知识点详解

### 2.1 自然光

**· 通俗解释：** 想象一群人向同一个方向奔跑，但每个人左右摇晃的方向是随机的、各方向机会均等。普通光源（如太阳、白炽灯）发出的光就是这样——光矢量在垂直于传播方向的平面内沿各个方向振动的几率均等，没有一个方向占优势。

**· 严格定义：** 在垂直于光波传播方向的平面内，光矢量沿各个方向振动的几率均等，且各方向之间无固定相位关系的光称为自然光。自然光可分解为两个互相垂直、振幅相等、彼此独立的线偏振光。

**· 数学表示：** 自然光的光强可表示为：
$$I_0 = I_{0x} + I_{0y}$$
其中 $I_{0x} = I_{0y} = \frac{1}{2}I_0$，即沿任意两个互相垂直方向的分振动强度相等。

**· 使用条件：** 普通光源（热光源）发出的光在通常条件下可视为自然光。

### 2.2 线偏振光（平面偏振光）

**· 通俗解释：** 想象一列人整齐地左右摆动，所有人摆动的方向完全一致。线偏振光就是光矢量始终沿一个固定方向振动的光。

**· 严格定义：** 光矢量始终在某一固定平面内沿某一固定方向振动的光称为线偏振光（或平面偏振光）。当迎着光的传播方向观察时，光矢量的振动方向为一条直线。

**· 使用条件：** 线偏振光可以通过偏振片、反射（布儒斯特角）或双折射等方法从自然光获得。

### 2.3 部分偏振光

**· 通俗解释：** 介于自然光和线偏振光之间——各个方向都有振动，但有一个方向振动更强一些。

**· 严格定义：** 光矢量沿各方向都有振动，但不同方向上的强度不同。部分偏振光可分解为一个自然光和一个线偏振光的叠加。

**· 使用条件：** 自然光经反射、折射或通过非理想偏振片后通常得到部分偏振光。

### 2.4 偏振片与起偏/检偏

**· 通俗解释：** 偏振片就像一道“光栅栏”——只允许振动方向与“栅栏方向”（偏振化方向）一致的光通过，其他方向的光被吸收。

**· 严格定义：** 利用晶体的二向色性（对相互垂直的光振动有选择吸收的性能）制成的光学器件称为偏振片。偏振片允许通过的光振动方向称为**偏振化方向**（或透振方向）。

**· 起偏：** 让自然光通过偏振片，透射光成为线偏振光，这个过程称为起偏，该偏振片称为起偏器。

**· 检偏：** 用偏振片检验光的偏振状态，该偏振片称为检偏器。

**· 使用条件：** 偏振片对透射光有吸收，实际应用中需考虑吸收损耗。

### 2.5 马吕斯定律

**· 通俗解释：** 线偏振光通过检偏器后，透射光的强弱取决于光的振动方向与检偏器透振方向的“对齐程度”——完全对齐时最强（全部透过），垂直时最弱（完全被挡住）。

**· 严格定义：** 光强为 $I_0$ 的线偏振光通过检偏器后，透射光强为：
$$I = I_0 \cos^2 \alpha$$
其中 $\alpha$ 是入射线偏振光的振动方向与检偏器偏振化方向之间的夹角。

**· 推导思路：** 设线偏振光的振幅为 $A_0$，振动方向与偏振片透振方向夹角为 $\alpha$。将光矢量分解为平行于透振方向的分量 $A_0 \cos\alpha$ 和垂直于透振方向的分量 $A_0 \sin\alpha$。偏振片只允许平行分量通过，故透射振幅 $A = A_0 \cos\alpha$。光强与振幅平方成正比，故 $I = I_0 \cos^2\alpha$。

**· 具体数值例子：** 一束光强为 $I_0 = 100 \text{ W/m}^2$ 的线偏振光垂直入射到检偏器上，其振动方向与检偏器透振方向夹角 $\alpha = 45^\circ$。求透射光强。

解：$I = I_0 \cos^2 45^\circ = 100 \times (\frac{\sqrt{2}}{2})^2 = 100 \times \frac{1}{2} = 50 \text{ W/m}^2$。

**· 使用条件/前提假设：** （1）入射光必须是线偏振光；（2）不考虑偏振片的吸收损耗；（3）适用于无吸收的理想偏振片。

**· 特殊情况：** 自然光通过偏振片后，不论如何旋转偏振片，透射光强恒为入射光强的一半。

### 2.6 布儒斯特定律

**· 通俗解释：** 当你以某个特定角度斜着看光滑玻璃表面时，反射光会变成“单向振动”的光（线偏振光）。这个特定角度就是布儒斯特角。

**· 严格定义：** 当自然光以特定入射角 $i_0$ 从介质1射向介质2的分界面时，反射光成为完全偏振光（振动方向垂直于入射面），这个入射角称为**布儒斯特角**（起偏角）。布儒斯特定律的数学表达式为：
$$\tan i_0 = \frac{n_2}{n_1} = n_{21}$$
其中 $n_1$、$n_2$ 分别为介质1和介质2的折射率，$n_{21}$ 是介质2对介质1的相对折射率。

当入射角为布儒斯特角时，反射光线与折射光线互相垂直。

**· 推导思路：** 由反射定律和折射定律，当反射光与折射光垂直时，$i_0 + r = 90^\circ$。由折射定律 $n_1 \sin i_0 = n_2 \sin r$，结合 $r = 90^\circ - i_0$，得 $n_1 \sin i_0 = n_2 \cos i_0$，即 $\tan i_0 = n_2/n_1$。

**· 具体数值例子：** 自然光从空气（$n_1 = 1.00$）入射到玻璃（$n_2 = 1.50$）表面，求布儒斯特角。

解：$\tan i_0 = \frac{n_2}{n_1} = \frac{1.50}{1.00} = 1.50$
$$i_0 = \arctan(1.50) \approx 56.3^\circ$$



**· 使用条件/前提假设：** （1）两种介质均为各向同性介质；（2）界面光滑；（3）反射光虽然是完全偏振光，但光强很弱（约15%）。

### 2.7 双折射现象

**· 通俗解释：** 把一根笔斜着插入水中，看起来像是“折断”了——这是光的折射。而双折射更神奇：一束光进入某些晶体（如方解石）后，会分裂成两束光，像分身术一样。

**· 严格定义：** 当光线进入某些各向异性晶体（如方解石、石英）时，折射光线分裂为两束，沿不同方向传播，这种现象称为**双折射**。

**· 两束光的名称与特性：**

|          | 寻常光（o光）        | 非常光（e光） |
| -------- | -------------------- | ------------- |
| 折射定律 | 遵守                 | 不遵守        |
| 折射率   | 对各方向为常量 $n_o$ | 与方向有关    |
| 振动方向 | 垂直于光轴           | 在光轴平面内  |

**· 光轴：** 晶体中不发生双折射的特殊方向，沿此方向o光和e光的折射率相等。

**· 使用条件：** 光入射到各向异性单轴晶体（如方解石、石英）时发生双折射。

### 2.8 波晶片（波片）

**· 通俗解释：** 波片就像一块“偏振态转换器”——线偏振光进去，出来的可能是圆偏振光、椭圆偏振光或旋转了方向的线偏振光，取决于波片的厚度和入射角度。

**· 严格定义：** 从单轴晶体中切割出的平行平面板，光轴与表面平行。线偏振光垂直入射后分解为o光和e光，两者通过波片后产生相位差：
$$\Delta = \frac{2\pi}{\lambda}(n_o - n_e)d$$
其中 $n_o$、$n_e$ 分别为o光和e光的折射率，$d$ 为波片厚度，$\lambda$ 为真空中的波长。

**· 常见波片类型：**

| 波片类型              | 相位差                       | 光程差                               | 作用                   |
| --------------------- | ---------------------------- | ------------------------------------ | ---------------------- |
| 四分之一波片（λ/4片） | $\Delta = \pm \frac{\pi}{2}$ | $(n_o-n_e)d = \pm \frac{\lambda}{4}$ | 线偏振光→圆/椭圆偏振光 |
| 半波片（λ/2片）       | $\Delta = \pm \pi$           | $(n_o-n_e)d = \pm \frac{\lambda}{2}$ | 线偏振光振动方向旋转   |
| 全波片（λ片）         | $\Delta = \pm 2\pi$          | $(n_o-n_e)d = \pm \lambda$           | 保持原偏振态           |

**· 具体数值例子：** 石英晶片对波长 $\lambda = 589 \text{ nm}$ 的光，$n_o = 1.544$，$n_e = 1.553$。求四分之一波片的最小厚度。

解：四分之一波片要求 $(n_o - n_e)d = \frac{\lambda}{4}$
$$d = \frac{\lambda}{4(n_o - n_e)} = \frac{589 \times 10^{-9}}{4 \times (1.553 - 1.544)} = \frac{589 \times 10^{-9}}{4 \times 0.009} \approx 1.64 \times 10^{-5} \text{ m} = 16.4 \text{ μm}$$

### 2.9 圆偏振光与椭圆偏振光

**· 通俗解释：** 线偏振光的光矢量像活塞一样来回直线振动；圆偏振光的光矢量像旋转木马一样匀速转动，端点画出一个圆；椭圆偏振光则像椭圆轨道，端点画出一个椭圆。

**· 严格定义：** 光矢量在垂直于传播方向的平面内，其大小和方向随时间变化，端点轨迹为椭圆（或圆）的光称为椭圆偏振光（或圆偏振光）。

**· 产生条件：** 线偏振光垂直入射到波片上，分解为o光和e光。设线偏振光振动方向与光轴夹角为 $\theta$，两分量的振幅分别为 $A_o = A\sin\theta$、$A_e = A\cos\theta$，相位差为 $\Delta$：
- 当 $\Delta = 0$ 或 $\pi$ 时 → 线偏振光
- 当 $\Delta = \frac{\pi}{2}$ 且 $\theta = 45^\circ$ 时 → 圆偏振光
- 当 $\Delta = \frac{\pi}{2}$ 且 $\theta \neq 45^\circ$ 时 → 椭圆偏振光
- 其他情况 → 椭圆偏振光

**· 左旋/右旋：** 迎着光传播方向观察，光矢量顺时针旋转称为右旋偏振光，逆时针旋转称为左旋偏振光。


## 3. 典型题型与完整解法

### 题型一：马吕斯定律——单偏振片或双偏振片光强计算

**· 题目特征：** 给出自然光或线偏振光的初始光强，经过一个或多个偏振片，求透射光强。关键词：“自然光入射”、“偏振片”、“偏振化方向夹角”、“透射光强”。

**· 标准解题步骤：**
1. 判断入射光的类型（自然光还是线偏振光）。
2. 若为自然光通过第一个偏振片，透射光强变为原来的一半，且变为线偏振光。
3. 之后每经过一个偏振片，用马吕斯定律 $I = I_0\cos^2\alpha$ 计算，其中 $\alpha$ 是入射光振动方向与该偏振片偏振化方向的夹角。
4. 注意：每经过一个偏振片后，光的振动方向变为该偏振片的偏振化方向。

**· 例题：** 光强为 $I_0$ 的自然光垂直通过两个偏振片 $P_1$ 和 $P_2$，$P_1$ 与 $P_2$ 的偏振化方向夹角为 $60^\circ$。求透射光强。

**解：**
- 自然光通过 $P_1$ 后：$I_1 = \frac{I_0}{2}$，变为线偏振光，振动方向沿 $P_1$ 的偏振化方向。
- 通过 $P_2$ 时，入射光振动方向与 $P_2$ 偏振化方向夹角 $\alpha = 60^\circ$。
- 由马吕斯定律：$I_2 = I_1\cos^2 60^\circ = \frac{I_0}{2} \times (\frac{1}{2})^2 = \frac{I_0}{8}$。

**答案：** $I = I_0/8$

**· 常见错误提醒：** （1）忘记自然光通过第一个偏振片后光强减半；（2）把自然光入射也直接套用 $\cos^2\alpha$；（3）搞错夹角——$\alpha$ 是光振动方向与偏振片透振方向的夹角，不是两个偏振片之间的几何夹角（若入射光已是线偏振光，两者等价）。

### 题型二：马吕斯定律——三偏振片问题

**· 题目特征：** 自然光依次通过三个偏振片，已知相邻偏振片之间的夹角，求最终透射光强，或求使透射光强最大的角度条件。

**· 标准解题步骤：**
1. 自然光通过第一片：$I_1 = I_0/2$，振动方向沿 $P_1$ 方向。
2. 通过第二片：$I_2 = I_1\cos^2\alpha_1$（$\alpha_1$ 为 $P_1$ 与 $P_2$ 夹角）。
3. 通过第三片：$I_3 = I_2\cos^2\alpha_2$（$\alpha_2$ 为 $P_2$ 与 $P_3$ 夹角）。
4. 若求最大光强，对角度求导或利用三角恒等式。

**· 例题：** 光强为 $I_0$ 的自然光依次通过三个偏振片 $P_1$、$P_2$、$P_3$。已知 $P_1 \perp P_3$（偏振化方向互相垂直），$P_2$ 与 $P_1$ 的夹角为 $\theta$。求透射光强 $I_3$，并求 $\theta$ 为何值时 $I_3$ 最大。

**解：**
- 通过 $P_1$：$I_1 = I_0/2$
- 通过 $P_2$：$I_2 = I_1\cos^2\theta = \frac{I_0}{2}\cos^2\theta$
- 通过 $P_3$：由于 $P_1 \perp P_3$，$P_2$ 与 $P_3$ 的夹角为 $90^\circ - \theta$
  $$I_3 = I_2\cos^2(90^\circ - \theta) = \frac{I_0}{2}\cos^2\theta \cdot \sin^2\theta = \frac{I_0}{8}\sin^2 2\theta$$

- 求最大值：$\sin^2 2\theta$ 最大为1，当 $2\theta = 90^\circ$，即 $\theta = 45^\circ$ 时，$I_{3,\max} = I_0/8$。

**答案：** $I_3 = \frac{I_0}{8}\sin^2 2\theta$，$\theta = 45^\circ$ 时最大

**· 常见错误提醒：** 注意 $P_1 \perp P_3$ 这个条件，不要错误地认为 $P_2$ 与 $P_3$ 的夹角就是 $\theta$。

### 题型三：布儒斯特定律——求布儒斯特角或折射率

**· 题目特征：** 给出两种介质的折射率，求布儒斯特角；或给出布儒斯特角，求折射率。关键词：“布儒斯特角”、“起偏角”、“反射光为完全偏振光”。

**· 标准解题步骤：**
1. 确定光是从哪种介质入射到哪种介质。
2. 套用公式 $\tan i_0 = n_2/n_1$。
3. 若要求折射率，则 $n_2 = n_1 \tan i_0$。
4. 注意：当入射角为布儒斯特角时，反射光与折射光垂直。

**· 例题：** 自然光从空气（$n_1 = 1.00$）射向某透明介质表面，测得当入射角为 $60^\circ$ 时反射光为完全偏振光。求该介质的折射率。

**解：**
- 反射光为完全偏振光 → 入射角等于布儒斯特角 $i_0 = 60^\circ$
- 由布儒斯特定律：$\tan i_0 = \frac{n_2}{n_1}$
- $n_2 = n_1 \tan 60^\circ = 1.00 \times \sqrt{3} \approx 1.73$

**答案：** $n_2 \approx 1.73$

**· 常见错误提醒：** （1）混淆 $n_2/n_1$ 和 $n_1/n_2$ 的顺序；（2）忘记反射光为完全偏振光时振动方向垂直于入射面；（3）折射光仍是部分偏振光，不是完全偏振光。

### 题型四：波片与偏振态转换

**· 题目特征：** 给出波片类型（λ/4或λ/2）和入射光的偏振态及方向，求透射光的偏振态。关键词：“四分之一波片”、“半波片”、“线偏振光”、“圆偏振光”。

**· 标准解题步骤：**
1. 确定入射光为线偏振光，将其分解为沿光轴方向和垂直光轴方向的两个分量。
2. 确定波片引入的相位差（λ/4片为π/2，λ/2片为π）。
3. 根据两个分量的振幅比和相位差，判断合成为何种偏振光。

**· 例题：** 线偏振光垂直入射到四分之一波片上，其振动方向与光轴夹角为 $45^\circ$。求透射光的偏振态。

**解：**
- 线偏振光分解为沿光轴（e光）和垂直光轴（o光）两个分量，振幅相等（因为 $\theta = 45^\circ$）：$A_e = A_o = A/\sqrt{2}$
- 四分之一波片使o光和e光产生 $\pi/2$ 的相位差
- 两个振幅相等、振动方向互相垂直、相位差为 $\pi/2$ 的振动合成 → **圆偏振光**

**答案：** 圆偏振光

**· 常见错误提醒：** （1）混淆o光和e光的振动方向；（2）忘记 $\theta = 45^\circ$ 是产生圆偏振光的必要条件（否则为椭圆偏振光）；（3）半波片使线偏振光振动方向旋转 $2\theta$（$\theta$ 为振动方向与光轴夹角），而不是 $\theta$。

### 题型五：混合光（自然光+线偏振光）通过偏振片

**· 题目特征：** 入射光是自然光和线偏振光的混合，旋转检偏器测得最大和最小透射光强，求两者的强度比或偏振度。

**· 标准解题步骤：**
1. 设自然光光强为 $I_n$，线偏振光光强为 $I_p$。
2. 自然光部分通过偏振片后恒为 $I_n/2$。
3. 线偏振光部分通过偏振片后为 $I_p\cos^2\alpha$，随 $\alpha$ 变化。
4. 最大透射光强：$I_{\max} = I_n/2 + I_p$（线偏振光与偏振片方向平行时）
5. 最小透射光强：$I_{\min} = I_n/2 + 0$（线偏振光与偏振片方向垂直时）
6. 偏振度：$P = \frac{I_{\max} - I_{\min}}{I_{\max} + I_{\min}}$

**· 例题：** 一束光是自然光和线偏振光的混合光，通过旋转检偏器测得最大透射光强是最小透射光强的5倍。求自然光和线偏振光的光强比。

**解：**
- 设自然光光强为 $I_n$，线偏振光光强为 $I_p$
- $I_{\max} = \frac{I_n}{2} + I_p$，$I_{\min} = \frac{I_n}{2}$
- 已知 $I_{\max} = 5I_{\min}$：
  $$\frac{I_n}{2} + I_p = 5 \times \frac{I_n}{2}$$
  $$I_p = 2I_n$$
  $$\frac{I_n}{I_p} = \frac{1}{2}$$

**答案：** $I_n : I_p = 1 : 2$

**· 常见错误提醒：** （1）忘记自然光部分通过偏振片后光强减半；（2）最小光强时线偏振光部分完全被挡住（$\alpha = 90^\circ$），而不是部分透过。


## 4. 易混淆概念对比表

### 对比组一：自然光 vs 线偏振光 vs 部分偏振光

| 对比项                 | 自然光                                   | 线偏振光                               | 部分偏振光                 |
| ---------------------- | ---------------------------------------- | -------------------------------------- | -------------------------- |
| **定义**               | 光矢量沿各方向振动几率均等               | 光矢量只沿一个固定方向振动             | 各方向都有振动，但强度不同 |
| **数学描述**           | 可分解为两个等幅、无固定相位差的垂直分量 | 只有一个振动方向                       | 可分解为自然光+线偏振光    |
| **通过偏振片后的光强** | 恒为入射光强的1/2                        | $I = I_0\cos^2\alpha$                  | 随旋转变化，但不消光       |
| **旋转偏振片观察**     | 光强不变                                 | 有消光现象（可完全变暗）               | 光强变化，但不消光         |
| **适用场景**           | 普通光源发出的光                         | 通过偏振片、反射（布儒斯特角）得到的光 | 自然光经反射、折射后的光   |
| **记忆技巧**           | “四面八方都一样”                         | “整齐划一一个方向”                     | “有主有次不均匀”           |

### 对比组二：寻常光（o光） vs 非常光（e光）

| 对比项       | 寻常光（o光）                              | 非常光（e光）                              |
| ------------ | ------------------------------------------ | ------------------------------------------ |
| **定义**     | 双折射中遵守折射定律的一束光               | 双折射中不遵守折射定律的一束光             |
| **折射率**   | 对各方向为常量 $n_o$                       | 与方向有关，沿光轴方向等于 $n_o$           |
| **振动方向** | 垂直于光轴                                 | 在光轴平面内                               |
| **传播速度** | 各方向相同                                 | 各方向不同                                 |
| **适用场景** | 光入射到单轴晶体（方解石、石英）时同时产生 | 光入射到单轴晶体（方解石、石英）时同时产生 |
| **记忆技巧** | “o”=ordinary=寻常=守规矩                   | “e”=extraordinary=非常=不守规矩            |

### 对比组三：四分之一波片 vs 半波片

| 对比项               | 四分之一波片（λ/4片）                    | 半波片（λ/2片）                    |
| -------------------- | ---------------------------------------- | ---------------------------------- |
| **定义**             | o、e光光程差为λ/4的波片                  | o、e光光程差为λ/2的波片            |
| **相位差**           | $\Delta = \pm \pi/2$                     | $\Delta = \pm \pi$                 |
| **对线偏振光的作用** | 线偏振光→圆偏振光（θ=45°时）或椭圆偏振光 | 线偏振光仍为线偏振光，振动方向旋转 |
| **对圆偏振光的作用** | 圆偏振光→线偏振光                        | 圆偏振光→圆偏振光（旋向改变）      |
| **适用场景**         | 需要在线偏振和圆偏振之间转换时           | 需要旋转线偏振光振动方向时         |
| **记忆技巧**         | “四分之一变圆”（45°入射线偏→圆偏）       | “二分之一转方向”（线偏方向旋转）   |


## 5. 本章小结与自测

### 一句话总结本章核心思想

**光是横波，其光矢量（电场强度E）的振动方向可以用偏振态来描述；通过偏振片（马吕斯定律）和界面反射（布儒斯特定律）可以获得并分析偏振光；利用晶体的双折射和波片可以实现不同偏振态之间的转换。**

### 自测题

#### 概念辨析题（2道）

**题1：** 一束光通过偏振片后，旋转偏振片一周，透射光强始终不变。这束光可能是（ ）
A. 线偏振光　　B. 自然光　　C. 圆偏振光　　D. 自然光或圆偏振光

**答案：D**

**解析：** 自然光通过偏振片后光强恒为入射光强的一半，不随偏振片旋转而变化。圆偏振光通过偏振片后光强也为恒定值（等于入射光强的一半）。线偏振光通过偏振片后光强随旋转变化且有消光现象。因此选D。

---

**题2：** 下列说法正确的是（ ）
A. 自然光以布儒斯特角入射到介质界面时，反射光和折射光都是完全偏振光
B. 自然光以布儒斯特角入射到介质界面时，反射光是完全偏振光，折射光是部分偏振光
C. 任何入射角下，反射光都是完全偏振光
D. 布儒斯特角的大小与波长有关

**答案：B**

**解析：** 自然光以布儒斯特角入射时，反射光是振动方向垂直于入射面的完全偏振光，但折射光仍是部分偏振光。A错在“折射光也是完全偏振光”。C错在“任何入射角下”——只有布儒斯特角时反射光才是完全偏振光。D错在布儒斯特角由折射率决定 $\tan i_0 = n_2/n_1$，而折射率与波长有关（色散），所以说“与波长无关”不准确；但通常在同一介质中讨论时，布儒斯特角主要取决于折射率。

#### 简单计算题（2道）

**题3：** 两偏振片的偏振化方向互相垂直，中间插入第三片偏振片，其偏振化方向与第一片成 $30^\circ$ 角。自然光强为 $I_0$ 垂直入射，求最终透射光强。

**答案：** $I = \frac{3I_0}{32}$

**解析：**
- 通过 $P_1$：$I_1 = I_0/2$
- 通过 $P_2$（与 $P_1$ 夹角 $30^\circ$）：$I_2 = I_1\cos^2 30^\circ = \frac{I_0}{2} \times \frac{3}{4} = \frac{3I_0}{8}$
- $P_3$ 与 $P_1$ 垂直，故 $P_2$ 与 $P_3$ 夹角为 $60^\circ$
- $I_3 = I_2\cos^2 60^\circ = \frac{3I_0}{8} \times \frac{1}{4} = \frac{3I_0}{32}$

**题4：** 自然光从空气（$n=1.00$）射向某液体表面，布儒斯特角为 $53^\circ$。求该液体的折射率。

**答案：** $n \approx 1.33$

**解析：** 由布儒斯特定律 $\tan i_0 = n_2/n_1$，$n_2 = n_1 \tan 53^\circ = 1.00 \times 1.327 \approx 1.33$。

#### 综合应用题（2道）

**题5：** 光强为 $I_0$ 的自然光依次通过三个偏振片 $P_1$、$P_2$、$P_3$。$P_1$ 与 $P_3$ 的偏振化方向夹角为 $90^\circ$，$P_2$ 与 $P_1$ 的偏振化方向夹角为 $\theta$。
（1）求透射光强 $I$ 与 $\theta$ 的关系式；
（2）当 $\theta$ 为何值时透射光强最大？最大光强是多少？

**答案：** （1）$I = \frac{I_0}{8}\sin^2 2\theta$；（2）$\theta = 45^\circ$，$I_{\max} = I_0/8$

**解析：**
- $I_1 = I_0/2$
- $I_2 = I_1\cos^2\theta = \frac{I_0}{2}\cos^2\theta$
- $P_3$ 与 $P_1$ 垂直，$P_2$ 与 $P_3$ 夹角为 $90^\circ - \theta$
- $I_3 = I_2\cos^2(90^\circ - \theta) = \frac{I_0}{2}\cos^2\theta\sin^2\theta = \frac{I_0}{8}\sin^2 2\theta$
- 最大值：$\sin^2 2\theta \leq 1$，当 $2\theta = 90^\circ$，即 $\theta = 45^\circ$ 时取最大值 $I_{\max} = I_0/8$

---

**题6：** 一束线偏振光垂直入射到四分之一波片上。已知波片光轴沿竖直方向。
（1）若入射线偏振光的振动方向与竖直方向成 $45^\circ$ 角，出射光是什么偏振态？
（2）若入射线偏振光的振动方向沿竖直方向，出射光是什么偏振态？
（3）若入射线偏振光的振动方向与竖直方向成 $30^\circ$ 角，出射光是什么偏振态？

**答案：** （1）圆偏振光；（2）线偏振光（不变）；（3）椭圆偏振光

**解析：**
- 线偏振光分解为沿光轴（竖直，e光）和垂直光轴（水平，o光）两个分量。
- （1）$\theta = 45^\circ$ 时两分量振幅相等，四分之一波片引入 $\pi/2$ 相位差 → 振幅相等、垂直、相位差 $\pi/2$ → **圆偏振光**。
- （2）$\theta = 0^\circ$ 时只有一个分量（沿光轴），没有两个分量干涉 → 出射仍是**线偏振光**，方向不变。
- （3）$\theta = 30^\circ$ 时两分量振幅不等（$A_e > A_o$），相位差 $\pi/2$ → **椭圆偏振光**。
