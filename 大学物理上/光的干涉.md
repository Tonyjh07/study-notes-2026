

# 大学物理 第7章 光的干涉 自学笔记


## 1. 本章知识地图

本章的核心是**光的干涉现象**——当两束或多束光波在空间相遇叠加时，光强在空间重新分布，形成明暗相间的条纹。

**知识的逻辑推进路径如下：**

**第一步：从“光是什么”出发。** 光的电磁理论证明光是一种电磁波，因此光波遵循波的叠加原理。但并非任意两列光波叠加都会产生干涉——只有满足特定条件的光波才能形成稳定的干涉图样。

**第二步：引入“相干条件”** ——频率相同、振动方向相同、相位差恒定。普通光源（如白炽灯、太阳）由大量原子随机发光，不满足相干条件，因此无法直接产生干涉。这就引出了核心问题：**如何获得相干光？**

**第三步：获得相干光的两种基本方法。** 方法一是**分波面法**（将同一波前分成两部分），代表实验是**杨氏双缝干涉**；方法二是**分振幅法**（将一束光的振幅分成两部分），代表现象是**薄膜干涉**。杨氏双缝干涉是理解干涉的起点，它直接给出了双缝干涉条纹的定量规律。

**第四步：为了处理光在不同介质中的传播问题，引入“光程”和“光程差”概念。** 光在介质中传播时波长会改变，但用“光程”（折射率×几何路程）可以将不同介质中的光程统一折算为真空中的路程，从而用统一的相位差公式处理。光程差是判断干涉加强/减弱的**核心物理量**。

**第五步：在光程差的基础上建立“干涉加强/减弱条件”** ——光程差等于波长的整数倍时加强（明纹），等于半波长的奇数倍时减弱（暗纹）。这个统一条件贯穿全章所有干涉问题。

**第六步：将干涉条件应用到具体场景中。** 薄膜干涉分为两类：**等倾干涉**（薄膜厚度均匀，不同入射角对应不同光程差）和**等厚干涉**（薄膜厚度不均匀，相同厚度处光程差相同）。等厚干涉的典型实例是**劈尖干涉**和**牛顿环**。

**第七步：介绍精密测量工具——迈克耳逊干涉仪。** 它利用分振幅干涉原理，通过移动一面镜子改变光程差，可精确测量长度、波长等物理量。

**整体逻辑链可概括为：** 光的波动性 → 相干条件 → 获得相干光的方法（分波面/分振幅）→ 光程与光程差 → 干涉加强/减弱条件 → 具体干涉装置（双缝/薄膜/劈尖/牛顿环/迈克耳逊干涉仪）→ 应用与测量。


## 2. 核心知识点详解

### 2.1 光的相干性

**· 通俗解释**

好比两列水波在水面上相遇——如果两列波的频率相同、振动方向一致、步调（相位）关系稳定，它们就会在某些点相互加强（波峰遇波峰），在另一些点相互抵消（波峰遇波谷），形成稳定的强弱分布图案。光的干涉也是同样道理。

**· 严格定义**

两列光波在空间相遇叠加后，若满足以下三个条件，则能产生稳定的干涉现象：

（1）**频率相同**（角频率 ω₁ = ω₂）

（2）**振动方向相同**（光矢量方向一致）

（3）**相位差恒定**（Δφ = φ₂ - φ₁ 不随时间变化）

满足这三个条件的光波称为**相干光**。

**· 为什么普通光源不满足相干条件**

普通光源（如白炽灯、太阳）的发光机制是**自发辐射**。每个原子独立、随机地发光，每次发光持续时间极短（小于 10⁻⁸ 秒），且不同原子发出的光波频率、振动方向、相位各不相同。因此普通光源发出的光**不相干**，不能直接产生干涉。

**· 获得相干光的方法**

既然普通光源不能直接产生干涉，就必须**从同一束光中分出两束相干光**。有两种基本方法：

- **分波面法**：将一个波前分割成两部分，这两部分来自同一波前，相位差恒定。代表：杨氏双缝干涉。
- **分振幅法**：将一束光的振幅（能量）分成两部分。代表：薄膜干涉。

**· 使用条件**

相干条件适用于所有类型的波（机械波、光波等）。对于光波，由于频率极高（约 10¹⁴ Hz），实际观测的是时间平均光强，因此相位差恒定是产生稳定干涉的关键。


### 2.2 杨氏双缝干涉

**· 通俗解释**

托马斯·杨在1801年做了一个著名的实验：让一束光先通过一个单缝（保证光来自同一波前），再通过两个紧挨着的狭缝。两缝相当于两个相干光源，发出的光在后面的屏幕上叠加，形成明暗相间的条纹——这是光具有波动性的最强证据。

**· 严格定义/公式**

**实验装置**：点光源 S → 单缝（产生相干光）→ 双缝 S₁、S₂（间距为 d）→ 观察屏（距双缝 D）。

**波程差**（两束光到达屏上 P 点的路程差）：
Δ = r₂ - r₁ ≈ d·x / D

其中：
- d —— 双缝间距
- D —— 双缝到屏的距离（D >> d）
- x —— P 点到屏中心 O 点的距离

**干涉加强（明纹）条件**：
Δ = k·λ （k = 0, 1, 2, ……）

**干涉减弱（暗纹）条件**：
Δ = (2k + 1)·λ/2 （k = 0, 1, 2, ……）

**明纹位置**：
x_k = k·λ·D / d

**暗纹位置**：
x_k = (2k + 1)·λ·D / (2d)

**相邻明（暗）条纹间距**：
Δx = λ·D / d

**· 推导思路**

1. **几何关系**：从 S₁ 和 S₂ 到屏上 P 点的距离分别为 r₁ 和 r₂。由于 D >> d，可近似认为两束光平行，波程差 Δ ≈ d·sinθ ≈ d·x/D。

2. **干涉条件**：两列相干光在 P 点的相位差由波程差决定：Δφ = 2π·Δ/λ。当 Δφ = 2kπ 时加强（明纹），Δφ = (2k+1)π 时减弱（暗纹）。

3. **条纹间距**：从第 k 级明纹到第 (k+1) 级明纹，x 增加 Δx = λD/d，说明条纹等间距分布。

**· 具体数值例子**

**例题**：在杨氏双缝干涉实验中，双缝间距 d = 0.50 mm，双缝到屏的距离 D = 1.2 m，用波长 λ = 600 nm（即 6.0×10⁻⁷ m）的单色光照射。求：（1）相邻明纹间距；（2）第3级明纹离屏中心多远？

**解**：

（1）相邻明纹间距：
Δx = λD/d = (6.0×10⁻⁷ × 1.2) / (0.50×10⁻³) = 1.44×10⁻³ m = 1.44 mm

（2）第3级明纹位置（k = 3）：
x₃ = 3λD/d = 3 × 1.44×10⁻³ = 4.32×10⁻³ m = 4.32 mm

**· 使用条件/前提假设**

1. D >> d（屏距远大于缝间距），否则近似不成立
2. 光源为单色光（若用白光，则彩色条纹）
3. 两缝为理想细缝，宽度远小于波长
4. 不考虑单缝衍射对光强的调制


### 2.3 光程与光程差

**· 通俗解释**

光在不同介质中传播时，速度不同，波长也会改变。但我们可以把光在介质中走的路程“折算”成在真空中走的路程——这个折算后的路程就叫“光程”。这样一来，不管光经过什么介质，我们都用统一的标准（真空中的波长）来计算相位变化。

**· 严格定义/公式**

**光程**：光在介质（折射率为 n）中传播的几何路程 r，折算到真空中的等效路程：
L = n · r

**光程差**：两束相干光的光程之差：
δ = L₁ - L₂ = n₁·r₁ - n₂·r₂

**相位差与光程差的关系**：
Δφ = 2π · δ / λ

其中 λ 为光在真空中的波长。

**干涉加强/减弱条件（用光程差表示）** ：

- 加强（明纹）：δ = k·λ（k = 0, 1, 2, ……）
- 减弱（暗纹）：δ = (2k + 1)·λ/2（k = 0, 1, 2, ……）

**· 推导思路**

1. 光在介质中的波长为 λₙ = λ/n（λ 为真空中波长）。
2. 光在介质中走 r 距离的相位变化为：Δφ = 2π·r/λₙ = 2π·n·r/λ。
3. 定义光程 L = n·r 后，相位变化统一为 Δφ = 2π·L/λ。
4. 两束光的相位差 = 2π·(L₁ - L₂)/λ = 2π·δ/λ。

**· 具体数值例子**

**例题**：在杨氏双缝干涉实验中，在其中一个缝后面放一块厚度 t = 1.0×10⁻⁵ m、折射率 n = 1.50 的透明薄片。用波长 λ = 550 nm 的光照射，求两束光在屏中心的光程差。

**解**：

不放薄片时，两束光到屏中心的光程相等（均为 D）。放薄片后，通过该缝的光在薄片中的光程为 n·t，而原本该段路程在空气中的光程为 t（空气折射率≈1）。

光程差：δ = (n - 1)·t = (1.50 - 1) × 1.0×10⁻⁵ = 5.0×10⁻⁶ m

**· 使用条件/前提假设**

1. 适用于任何透明介质
2. 光程是几何路程与折射率的乘积，不是实际物理路程
3. 注意半波损失（光从光疏介质射向光密介质反射时，反射光有 π 的相位突变，等效于附加 λ/2 的光程）


### 2.4 薄膜干涉——等倾干涉与等厚干涉

**· 通俗解释**

肥皂泡表面五颜六色、油膜上的彩色花纹，都是薄膜干涉的日常例子。光照射到薄膜的上表面时，一部分光直接反射，另一部分光透射进入薄膜、在下表面反射后再穿出——这两束反射光来自同一入射光，是相干光，叠加后产生干涉。

**· 薄膜干涉的基本公式**

设薄膜折射率为 n，厚度为 e，入射角为 i，折射角为 γ。

**反射光的光程差**：
δ = 2·n·e·cosγ + λ/2

其中 λ/2 项来自**半波损失**：光从光疏介质射向光密介质（n₁ < n₂）的反射面上反射时，反射光有 π 的相位突变，等效于附加 λ/2 的光程。

**透射光的光程差**：
δ = 2·n·e·cosγ

（透射光无半波损失）

**干涉条件**：
- 明纹：δ = k·λ（k = 1, 2, 3, ……）
- 暗纹：δ = (2k + 1)·λ/2（k = 0, 1, 2, ……）

**· 等倾干涉 vs 等厚干涉**

|                      | **等倾干涉**                           | **等厚干涉**                     |
| -------------------- | -------------------------------------- | -------------------------------- |
| **薄膜特征**         | 厚度均匀（e = 常数）                   | 厚度不均匀（e 变化）             |
| **决定光程差的因素** | 入射角 i（倾角）                       | 薄膜厚度 e                       |
| **条纹形状**         | 同心圆环                               | 平行于棱边的直条纹               |
| **观察方式**         | 用扩展光源，在无穷远处或透镜焦平面观察 | 用平行光垂直入射，在薄膜表面观察 |
| **典型例子**         | 厚度均匀的肥皂膜                       | 劈尖、牛顿环                     |

**· 使用条件/前提假设**

1. 薄膜上下表面平行（等倾）或近似平行（等厚，夹角很小）
2. 只考虑单次反射（薄膜较薄，多次反射可忽略）
3. 半波损失的有无取决于薄膜两侧介质的折射率关系


### 2.5 劈尖干涉

**· 通俗解释**

两块平板玻璃一端接触、另一端垫一片薄物，就形成一个楔形的空气层——“劈尖”。用单色光垂直照射时，空气层上下表面的反射光发生干涉，形成一系列平行于棱边的等间距直条纹。条纹的疏密可以反映出劈尖的倾斜角度。

**· 严格定义/公式**

两块玻璃板之间的空气薄膜构成劈尖，劈尖角为 θ（很小）。

**光程差**（正入射，空气劈尖 n₂ = 1）：
δ = 2e + λ/2

（其中 λ/2 来自光在玻璃-空气界面的半波损失）

**明纹条件**：
2e + λ/2 = k·λ → e = (2k - 1)·λ/4 （k = 1, 2, 3, ……）

**暗纹条件**：
2e + λ/2 = (2k + 1)·λ/2 → e = k·λ/2 （k = 0, 1, 2, ……）

**相邻明（暗）纹对应的厚度差**：
Δe = λ/2

**相邻条纹间距**：
l = Δe / sinθ ≈ λ / (2θ)

（θ 越小，条纹间距越大，条纹越稀；θ 越大，条纹越密）

**· 推导思路**

1. 劈尖上任一点处的空气膜厚度 e 与该点到棱边的距离 x 有关：e = x·sinθ ≈ x·θ（θ 很小）。
2. 代入光程差公式 δ = 2e + λ/2。
3. 由明纹/暗纹条件解出 e，相邻条纹的 e 相差 λ/2。
4. 由几何关系 l = Δe/θ = λ/(2θ)。

**· 具体数值例子**

**例题**：用波长 λ = 589 nm 的钠黄光垂直照射空气劈尖，测得相邻暗纹间距 l = 1.5 mm。求劈尖角 θ。

**解**：
l = λ/(2θ) → θ = λ/(2l) = 589×10⁻⁹ / (2 × 1.5×10⁻³) = 1.96×10⁻⁴ rad

**· 使用条件/前提假设**

1. 正入射（光线垂直照射）
2. 劈尖角 θ 很小（sinθ ≈ θ）
3. 空气劈尖（n₂ = 1），其他介质需代入相应折射率


### 2.6 牛顿环

**· 通俗解释**

将一个曲率半径很大的平凸透镜放在一块平板玻璃上，两者之间就形成一个**厚度从中心向外逐渐增加**的空气薄膜。用单色光垂直照射时，从上方观察会看到一组**同心圆环**——中心是暗点，向外是明暗相间的圆环，环间距从中心向外越来越密。牛顿最早发现了这一现象，因此得名。

**· 严格定义/公式**

**几何关系**：设透镜曲率半径为 R，距接触点 O 为 r 处的空气膜厚度为 e：
r² = R² - (R - e)² = 2Re - e²

由于 e << R（R 通常很大），e² 可忽略：
e ≈ r² / (2R)

**光程差**（正入射，空气膜 n = 1，有半波损失）：
δ = 2e + λ/2 = r²/R + λ/2

**明环半径**：
r_k = √[(2k - 1)·R·λ / 2] （k = 1, 2, 3, ……）

**暗环半径**：
r_k = √(k·R·λ) （k = 0, 1, 2, 3, ……）

**· 推导思路**

1. 由几何关系得到 e ≈ r²/(2R)。
2. 代入光程差 δ = 2e + λ/2 = r²/R + λ/2。
3. 明纹条件 δ = kλ → r²/R + λ/2 = kλ → r = √[(2k-1)Rλ/2]。
4. 暗纹条件 δ = (2k+1)λ/2 → r²/R + λ/2 = (2k+1)λ/2 → r = √(kRλ)。

**· 具体数值例子**

**例题**：用波长 λ = 589 nm 的钠黄光观察牛顿环，测得第5级暗环半径 r₅ = 3.0 mm。求平凸透镜的曲率半径 R。

**解**：
暗环半径公式：r_k = √(k·R·λ)
R = r_k² / (k·λ) = (3.0×10⁻³)² / (5 × 589×10⁻⁹) = 9.0×10⁻⁶ / 2.945×10⁻⁶ ≈ 3.06 m

**· 使用条件/前提假设**

1. 透镜曲率半径 R 远大于环半径 r（e << R）
2. 正入射（光垂直照射）
3. 空气薄膜（若充其他介质，n ≠ 1，需修正公式）


### 2.7 迈克耳逊干涉仪

**· 通俗解释**

迈克耳逊干涉仪利用**分振幅法**产生干涉。一束光被一块半透半反镜分成两束——一束反射、一束透射——分别射向两块互相垂直的平面镜 M₁ 和 M₂，反射后回到半透半反镜会合，产生干涉。通过移动其中一面镜子改变光程差，可以精确测量微小长度变化。

**· 核心原理**

设 M₁ 和 M₂ 到半透半反镜的距离分别为 d₁ 和 d₂。两束相干光的光程差为：
δ = 2(d₁ - d₂) = 2Δd

当 M₁ 移动 Δd 时，光程差变化 2Δd，条纹移动 N 条：
2Δd = N·λ

由此可测量微小位移或波长。

**· 使用条件/前提假设**

1. M₁ 和 M₂ 互相垂直（产生等倾干涉圆环）或稍有夹角（产生等厚干涉直条纹）
2. 光源为单色光


## 3. 典型题型与完整解法

### 题型一：杨氏双缝干涉基础计算

**· 题目特征**

已知双缝间距 d、缝屏距离 D、波长 λ，求条纹间距、某级条纹位置、条纹数等。关键词：“双缝干涉”、“明纹”、“暗纹”、“条纹间距”。

**· 标准解题步骤**

1. 写出波程差公式：Δ = d·x/D
2. 根据所求（明纹/暗纹）代入相应条件：
   - 明纹：Δ = kλ → x = kλD/d
   - 暗纹：Δ = (2k+1)λ/2 → x = (2k+1)λD/(2d)
3. 条纹间距：Δx = λD/d（明纹和暗纹间距相同）
4. 代入数据计算

**· 完整例题**

**题干**：在杨氏双缝干涉实验中，双缝间距 d = 0.20 mm，屏距 D = 1.0 m，用波长 λ = 500 nm 的单色光垂直照射。（1）求相邻明纹间距；（2）屏上共能看到多少条明纹？（设双缝宽度远小于波长，忽略衍射调制）

**解答**：

（1）相邻明纹间距：
Δx = λD/d = (500×10⁻⁹ × 1.0) / (0.20×10⁻³) = 2.5×10⁻³ m = 2.5 mm

（2）屏上可见明纹范围取决于干涉条纹的角宽度。设屏宽度不限，明纹位置 x_k = kλD/d。当 k 增大到使 sinθ ≈ 1 时（θ = 90°），Δ = d ≈ kλ，k_max ≈ d/λ = 0.20×10⁻³ / (500×10⁻⁹) = 400。但实际屏的有限宽度会限制可见条纹数。若屏宽为 L，则 |k| ≤ Ld/(2λD)。（本题若未给屏宽，通常答“中央明纹两侧各 k_max 条，共 2k_max+1 条”）

**答案**：（1）2.5 mm；（2）若屏足够宽，共801条（k = -400 到 400）

**· 常见错误提醒**

- ❌ 混淆明纹和暗纹的条件（明纹 k 从0开始，暗纹 k 从0开始但条件是半波长的奇数倍）
- ❌ 忘记单位换算（mm→m，nm→m）
- ❌ 把 D 和 d 弄反


### 题型二：光程差与介质插入问题

**· 题目特征**

在双缝或干涉装置中插入介质薄片（玻璃片、薄膜等），问条纹如何移动、光程差变化多少。关键词：“覆盖玻璃片”、“插入介质”、“条纹移动”、“折射率”。

**· 标准解题步骤**

1. 计算插入介质引起的光程差变化：Δδ = (n - 1)·t（t 为介质厚度）
2. 条纹移动级数：N = Δδ / λ
3. 判断移动方向：光程增加的那一侧，条纹向该侧移动
4. 若问某点变为第几级明纹，用 δ = kλ 求解

**· 完整例题**

**题干**：在杨氏双缝干涉实验中，用波长 λ = 550 nm 的单色光照射。在其中一个缝后面放一块厚度 t = 4.8×10⁻⁶ m、折射率 n = 1.58 的玻璃片。问：（1）屏中心 O 点现在是明纹还是暗纹？（2）零级明纹移到了原来第几级明纹的位置？

**解答**：

（1）光程差变化：
Δδ = (n-1)t = (1.58 - 1) × 4.8×10⁻⁶ = 2.784×10⁻⁶ m
Δδ / λ = 2.784×10⁻⁶ / (550×10⁻⁹) = 5.06 ≈ 5

所以 O 点的光程差 δ ≈ 5λ，是**明纹**（第5级）。

（2）零级明纹（δ = 0）移到原来光程差为 5λ 的位置，即原来第5级明纹处。

**答案**：（1）明纹；（2）第5级明纹处

**· 常见错误提醒**

- ❌ 光程差变化是 (n-1)t，不是 nt
- ❌ 忘记判断半波损失（若反射情况需考虑 λ/2）
- ❌ 条纹移动方向判断错误


### 题型三：薄膜干涉（增透膜/增反膜）

**· 题目特征**

求镀膜的最小厚度、判断反射光加强还是减弱。关键词：“增透膜”、“增反膜”、“镜头镀膜”、“反射光相消”。

**· 标准解题步骤**

1. 判断半波损失：光从折射率 n₁→n₂（n₁ < n₂）反射时有半波损失
2. 写出光程差公式（含或不含 λ/2）
3. 增透膜要求反射光相消（暗纹）：δ = (2k+1)λ/2
4. 增反膜要求反射光相长（明纹）：δ = kλ
5. 取 k = 1（最小厚度）或 k = 0 代入求解

**· 完整例题**

**题干**：照相机的镜头（n₃ = 1.52）上镀了一层氟化镁薄膜（n₂ = 1.38），目的是减少反射（增透膜）。用波长 λ = 550 nm 的可见光垂直照射，求膜的最小厚度。

**解答**：

光从空气（n₁ = 1）→ 氟化镁（n₂ = 1.38）→ 玻璃（n₃ = 1.52）。n₁ < n₂ < n₃，上下两个反射面都有半波损失，两束反射光总的半波损失相互抵消，**无附加半波损失**。

光程差：δ = 2·n₂·e（正入射，cosγ = 1）

增透膜要求反射光相消（暗纹）：
2·n₂·e = (2k + 1)·λ/2

取 k = 0（最小厚度）：
e = λ / (4·n₂) = 550×10⁻⁹ / (4 × 1.38) = 9.96×10⁻⁸ m ≈ 99.6 nm

**答案**：99.6 nm

**· 常见错误提醒**

- ❌ 半波损失判断错误——必须逐层分析折射率大小关系
- ❌ 增透膜是反射光相消（暗纹），不是相长
- ❌ 忘记取 k = 0 还是 k = 1（看题目要求“最小厚度”通常取 k = 0）


### 题型四：劈尖干涉

**· 题目特征**

已知劈尖角、波长、条纹间距等，求未知量。关键词：“劈尖”、“楔形薄膜”、“条纹间距”、“空气劈尖”。

**· 标准解题步骤**

1. 判断半波损失（空气劈尖：光在玻璃-空气界面反射有半波损失）
2. 写出光程差：δ = 2e + λ/2（空气劈尖）
3. 条纹间距公式：l = λ/(2θ)
4. 若求厚度变化：Δe = λ/2

**· 完整例题**

**题干**：用波长 λ = 589 nm 的钠黄光垂直照射空气劈尖，测得10条暗纹的总宽度为6.0 mm。求劈尖角 θ。

**解答**：

10条暗纹之间有9个间距，每个间距 l = λ/(2θ)。

总宽度：9l = 9×λ/(2θ) = 6.0×10⁻³ m

θ = 9λ / (2 × 6.0×10⁻³) = (9 × 589×10⁻⁹) / (1.2×10⁻²) = 4.42×10⁻⁴ rad

**答案**：4.42×10⁻⁴ rad

**· 常见错误提醒**

- ❌ N 条条纹有 (N-1) 个间距，不是 N 个
- ❌ 忘记半波损失中的 λ/2 项
- ❌ 角度单位混淆（rad 与度）


### 题型五：牛顿环

**· 题目特征**

已知牛顿环某级暗（明）环半径、波长，求曲率半径；或已知 R、λ 求环半径。关键词：“牛顿环”、“平凸透镜”、“曲率半径”、“暗环半径”。

**· 标准解题步骤**

1. 判断是明环还是暗环
2. 选择相应公式：
   - 明环：r = √[(2k-1)·R·λ / 2]
   - 暗环：r = √(k·R·λ)
3. 代入数据求解

**· 完整例题**

**题干**：用波长 λ = 650 nm 的红光观察牛顿环，测得第10级暗环半径 r₁₀ = 4.5 mm。求平凸透镜的曲率半径 R。若换用波长 λ' = 450 nm 的蓝光，同一位置的环的级次变为多少？

**解答**：

（1）暗环公式：r_k = √(k·R·λ)
R = r₁₀² / (10λ) = (4.5×10⁻³)² / (10 × 650×10⁻⁹) = 2.025×10⁻⁵ / 6.5×10⁻⁶ ≈ 3.12 m

（2）换用蓝光后，同一位置 r 不变：
r² = k·R·λ = k'·R·λ'
k' = k · (λ / λ') = 10 × (650 / 450) ≈ 14.4

因为 k 必须为整数，第14级暗环在 r 略小处，第15级暗环在 r 略大处。

**答案**：（1）3.12 m；（2）约第14～15级之间

**· 常见错误提醒**

- ❌ 明环和暗环公式搞混（明环有 (2k-1)，暗环有 k）
- ❌ 忘记 k 的起始值（暗环 k = 0,1,2,…；明环 k = 1,2,3,…）
- ❌ 中心点是暗点（不是亮点）——因为中心处 e = 0，δ = λ/2，满足暗纹条件


## 4. 易混淆概念对比表

### 对比组一：等倾干涉 vs 等厚干涉

|                    | **等倾干涉**                                   | **等厚干涉**                             |
| ------------------ | ---------------------------------------------- | ---------------------------------------- |
| **定义**           | 厚度均匀的薄膜，不同入射角（倾角）的光产生干涉 | 厚度不均匀的薄膜，相同厚度处产生同级干涉 |
| **决定光程差的量** | 入射角 i（倾角）                               | 薄膜厚度 e                               |
| **条纹形状**       | 同心圆环                                       | 平行直条纹（劈尖）或同心圆环（牛顿环）   |
| **光源要求**       | 扩展光源                                       | 平行光（点光源在透镜焦平面）             |
| **观察位置**       | 透镜焦平面或无穷远处                           | 薄膜表面附近                             |
| **典型装置**       | 厚度均匀的肥皂膜、平行平板                     | 劈尖、牛顿环                             |
| **记忆技巧**       | “等倾看角度，一圈一个角”                       | “等厚看厚度，一条一个厚”                 |

### 对比组二：明纹条件 vs 暗纹条件

|                  | **明纹（干涉加强）**       | **暗纹（干涉减弱）**                         |
| ---------------- | -------------------------- | -------------------------------------------- |
| **光程差条件**   | δ = kλ（k = 0, ±1, ±2, …） | δ = (2k+1)λ/2（k = 0, ±1, ±2, …）            |
| **相位差条件**   | Δφ = 2kπ                   | Δφ = (2k+1)π                                 |
| **物理意义**     | 两列波同相叠加，振幅相加   | 两列波反相叠加，振幅相减                     |
| **杨氏双缝位置** | x = kλD/d                  | x = (2k+1)λD/(2d)                            |
| **中央条纹**     | 中央为明纹（k = 0）        | 中央无暗纹（k = 0 时 x = λD/(2d)，不在中心） |
| **记忆技巧**     | “整波长为明，半波奇数为暗” |                                              |

### 对比组三：分波面法 vs 分振幅法

|                | **分波面法**                   | **分振幅法**                     |
| -------------- | ------------------------------ | -------------------------------- |
| **原理**       | 将同一波前分割成两部分         | 将一束光的振幅（能量）分成两部分 |
| **相干性来源** | 来自同一波前，相位差恒定       | 来自同一入射光，相位差恒定       |
| **典型装置**   | 杨氏双缝、菲涅耳双镜、劳埃德镜 | 薄膜干涉、迈克耳逊干涉仪         |
| **光强特点**   | 两束光强相近，条纹清晰         | 反射光和透射光光强不同           |
| **应用场景**   | 实验室观察干涉现象             | 实际光学器件（增透膜、干涉仪）   |
| **记忆技巧**   | “波面分两半，双缝是典范”       | “振幅分两份，薄膜是典型”         |


## 5. 本章小结与自测

### 一句话总结

> **光的干涉本质上是相干光波叠加导致光强重新分布的现象，其核心是光程差——光程差为波长整数倍时加强（明纹），为半波长奇数倍时减弱（暗纹），这一统一条件贯穿杨氏双缝、薄膜、劈尖、牛顿环等所有干涉问题。**


### 自测题

#### 一、概念辨析题（2道）

**题1**：下列说法是否正确？请判断并说明理由：“只要两列光波频率相同，相遇时就一定能产生干涉条纹。”

**答案**：错误。两列光波要产生稳定的干涉条纹，必须同时满足三个条件：频率相同、振动方向相同、相位差恒定。仅频率相同不够，如果振动方向垂直或相位差随机变化，不会产生稳定的干涉图样。

**题2**：在薄膜干涉中，为什么反射光干涉加强时，透射光干涉反而减弱？

**答案**：这是**能量守恒**的结果。入射光的能量被反射光和透射光分配。当反射光因干涉而加强时，说明反射光的能量增大；在无吸收的理想情况下，总能量守恒，透射光的能量必然相应减小，表现为透射光干涉减弱。

#### 二、简单计算题（2道）

**题3**：在杨氏双缝干涉实验中，d = 0.30 mm，D = 1.5 m，用 λ = 480 nm 的蓝光照射。求：（1）相邻明纹间距；（2）第4级暗纹离屏中心多远。

**答案**：

（1）Δx = λD/d = 480×10⁻⁹×1.5 / (0.30×10⁻³) = 2.4×10⁻³ m = 2.4 mm

（2）暗纹位置：x = (2k+1)λD/(2d)，k = 4 时：
x₄ = (2×4+1)×480×10⁻⁹×1.5 / (2×0.30×10⁻³) = 9×480×10⁻⁹×1.5 / (6×10⁻⁴) = 1.08×10⁻² m = 10.8 mm

**题4**：空气劈尖的劈尖角 θ = 2.0×10⁻⁴ rad，用波长 λ = 550 nm 的光垂直照射，求相邻明纹间距。

**答案**：
l = λ/(2θ) = 550×10⁻⁹ / (2×2.0×10⁻⁴) = 1.375×10⁻³ m = 1.375 mm

#### 三、综合应用题（2道）

**题5**：在杨氏双缝干涉实验中，若将整个装置浸入水中（水的折射率 n = 1.33），其他条件（d、D、光源频率）不变。问：（1）条纹间距变为原来的多少倍？（2）若在水中仍能看到干涉条纹，需要满足什么条件？

**答案**：

（1）光在水中的波长变为 λ' = λ/n（λ 为真空中波长）。条纹间距 Δx' = λ'D/d = (λ/n)D/d = Δx/n。所以条纹间距变为原来的 1/1.33 ≈ 0.75 倍，条纹变密。

（2）光从空气进入水中，光源频率不变，但波长减小。只要光源的相干长度足够（普通光源的相干长度仅约 1 mm～10 cm），双缝到屏的距离 D 不能太大，否则光程差超过相干长度，条纹消失。

**题6**：用牛顿环实验测量透镜曲率半径。测得第5级暗环半径 r₅ = 2.8 mm，第10级暗环半径 r₁₀ = 4.2 mm，波长 λ = 589 nm。求曲率半径 R。

**答案**：

暗环公式：r_k² = k·R·λ

联立：
r₅² = 5Rλ
r₁₀² = 10Rλ

两式相减：r₁₀² - r₅² = 5Rλ

R = (r₁₀² - r₅²) / (5λ)
= [(4.2×10⁻³)² - (2.8×10⁻³)²] / (5 × 589×10⁻⁹)
= (1.764×10⁻⁵ - 7.84×10⁻⁶) / (2.945×10⁻⁶)
= 9.80×10⁻⁶ / 2.945×10⁻⁶
≈ 3.33 m

**简要解析**：用两级暗环半径的平方差求 R，可以消除中心接触点处可能存在的灰尘或形变带来的系统误差，是实验中常用的方法。
